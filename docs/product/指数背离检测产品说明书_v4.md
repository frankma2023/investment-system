# 指数背离检测产品说明书 v4

> 状态：v4（定稿），待开发  
> 对应引擎：`src/detectors/divergence.py`  
> 数据库表：`index_divergence_daily`

## 1. 产品概述

指数背离检测用于识别行情走势与技术指标之间的不一致，作为趋势衰竭的**预警信号**。背离本身不是买卖信号，需与其他模块（RS强度、拥挤度、A/D评级）综合使用。

**核心原则：** 四种背离独立展示，不做加权合并。支持灵敏度切换（短线/长线）及量价模式切换（快速/慢速）。盘后日频计算，结果持久化。

## 2. 背离类型

### 2.1 量价背离（P0）

| 模式 | 计算 | 适用 |
|------|------|------|
| 快速（点对点） | 今日收盘/量 vs 昨日 | 短期异动 |
| 慢速（均线趋势） | MA5 vs 10日前MA5 | 中长线趋势 |

**共用逻辑：** 价格向上+量能向下=顶背离，反之=底背离。强度=|价格变动%|+|量变动%|，<3%不报警。

| 参数 | 快速默认 | 慢速默认 |
|------|---------|---------|
| MA周期 | 1 | 5日 |
| 对比回看N | 1日 | 10日 |
| 最低强度 | 3% | 3% |
| 有效期 | 10日 | 10日 |

### 2.2 RSI背离（P0）

- v1用固定阈值70/30。动态分位作为"高级"选项同时可用。
- 用户手动选择后参数保持，不随灵敏度切换重置。

**顶背离：** 价格>过去N日高点 且 RSI<过去N日RSI峰值 且 RSI>超买线(70)  
**底背离：** 价格<过去N日低点 且 RSI>过去N日RSI谷值 且 RSI<超卖线(30)

| 参数 | 短线默认 | 长线默认 |
|------|---------|---------|
| RSI周期 | 7日 | 14日 |
| N窗口 | 10日 | 20日 |
| 超买/超卖 | 70/30 | 70/30 |
| 动态分位(高级) | 90%/10% | 90%/10% |

### 2.3 MACD背离（P0）

标准MACD(12/26/9)。顶背离需MACD柱>0，底背离需MACD柱<0。

| 参数 | 短线默认 | 长线默认 |
|------|---------|---------|
| 快/慢/信号线 | 12/26/9 | 12/26/9 |
| N窗口 | 10日 | 20日 |
| 有效期 | 10日 | 10日 |

### 2.4 成分股上涨比例背离（P1）

- 全市场/L1/L2行业（86个指数）：全成分股计算
- 主题/策略（322个指数）：前20大权重股

**顶背离：** 今日涨 且 上涨比例<昨日 且 上涨比例<50%，连续2日  
**底背离：** 今日跌 且 上涨比例>昨日 且 上涨比例<40%，连续2日

| 参数 | 默认 |
|------|------|
| 连续天数 | 2日 |
| 顶背离阈值 | 50% |
| 底背离阈值 | 40% |
| 有效期 | 5日 |

## 3. 信号等级（统一）

| 等级 | 条件 |
|------|------|
| ⚠️ 潜在 | 背离首次出现，价格尚未确认 |
| 🔴 确认 | 顶背离+跌破近20日低点；或 底背离+突破近20日高点 |
| 🔴🔴 强烈 | 确认+均线交叉(5日线穿10日线) |

## 4. 信号衰减与自动延展

- 有效期10日（成分股5日），超期未确认自动清除
- 自动延展：信号触达但未确认，价格继续向背离方向运动→延5日（最多一次）

## 5. 共振强度判定规则

### 5.0 各模块极端信号映射

| 模块 | 看多信号 | 中性 | 看空信号 |
|------|---------|------|---------|
| RS强度 | A+, A | B, C+ | C, D, E |
| A/D评级 | A+, A | B, C+ | C, D, E |
| 拥挤度 | 低拥挤, 正常 | 偏高 | 高拥挤 |

### 5.1 共振强度等级

| 等级 | 条件 |
|------|------|
| **高** | ≥2个看空信号 + 任一类顶背离 = "高共振 · 强烈回调风险" |
| **高** | ≥2个看多信号 + 任一类底背离 = "高共振 · 强烈反转机会" |
| **中** | 1个极端信号 + 背离，或 背离同时发生在两种指标上 |
| **低** | 仅背离，无其他模块极端信号 |

### 5.2 综合预警文字

系统自动生成一行说明，例如：
- "RS强度A+，拥挤度高，MACD顶背离 → 高共振 · 强烈回调风险"
- "A/D评级E，RSI底背离 → 中共振 · 关注反转"
- "量价顶背离 → 低共振"

## 6. 灵敏度选择

两个Radio按钮：「短线（敏感）」「长线（稳健）」。切换时自动更新所有参数（RSI周期/N窗口/量价模式/确认窗口），但**每个参数保持独立可调**，用户可手动微调。

| 模式 | RSI周期 | N窗口 | 量价模式 | 确认窗口 | 有效期 |
|------|---------|-------|----------|----------|--------|
| 短线 | 7日 | 10日 | 快速 | 10日 | 10日 |
| 长线 | 14日 | 20日 | 慢速 | 20日 | 10日+延展 |

## 8. 技术实现

### 8.1 数据库表

```sql
CREATE TABLE index_divergence_daily (
    stock_code TEXT NOT NULL,
    date TEXT NOT NULL,
    -- 量价背离
    div_vp_type TEXT,       -- top / bottom / null
    div_vp_level INTEGER,   -- 1潜在/2确认/3强烈
    div_vp_strength REAL,   -- 背离强度
    -- RSI背离
    div_rsi_type TEXT,
    div_rsi_level INTEGER,
    -- MACD背离
    div_macd_type TEXT,
    div_macd_level INTEGER,
    -- 成分股背离
    div_breadth_type TEXT,
    div_breadth_level INTEGER,
    -- 共振
    resonance_level TEXT,   -- 高/中/低
    alert_text TEXT,
    -- 快照数据
    close REAL,
    rs_rating TEXT,
    ad_rating TEXT,
    crowd_level TEXT,
    updated_at TEXT DEFAULT (datetime('now','localtime')),
    PRIMARY KEY (stock_code, date)
);
```

### 8.2 API

```
GET /api/index-divergence?pool=market&date=2026-05-06&sensitivity=long&volume_mode=slow
```

### 8.4 计算流程

```
每日盘后（约16:00）：
  1. 拉取408个指数日线 + 成分股日线
  2. 逐个指数计算四种背离
  3. 查询RS/AD/拥挤度评级（来自已有模块）
  4. 计算共振等级 + 生成预警文字
  5. 写入 index_divergence_daily 表
```
