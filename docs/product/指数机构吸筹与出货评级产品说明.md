# 产品说明文档：指数机构吸筹/出货评级（A/D Rating for Indices）

> 对应引擎：`src/detectors/index_ad.py`  
> 配置文件：`config/index_ad.yaml`  
> 最后更新：2026-05-06（与实际开发对齐）

## 1. 概述

本产品将IBD（Investor's Business Daily）体系中经典的个股机构吸筹/出货评级（Accumulation/Distribution Rating）迁移至指数层面，用以衡量机构资金在某一指数（宽基、行业、主题、策略）上的近期净流入/流出程度。

## 2. 数据来源

| 数据项 | 来源 | 字段 | 说明 |
|--------|------|------|------|
| 指数日线行情 | index_daily_kline | open, high, low, close | 每日OHLC |
| 指数成交金额 | index_daily_kline | amount | 用于计算换手率 |
| 换手率 | index_fundamental_daily | to_r | 直接换手率（仅6个指数有值） |
| 总市值 | index_fundamental_daily | mc | to_r缺失时用于兜底计算 |

> 换手率取值：优先使用 `to_r`，若为NULL则自动计算 `amount / mc`。  
> raw方法需至少65个交易日历史，zscore方法需至少315个交易日（250天基线 + 65天窗口）。

## 3. 计算方法

### 3.1 日级别资金流量（Money Flow Value, MFV）

```
MFV = [(收盘价 - 最低价) - (最高价 - 收盘价)] / (最高价 - 最低价) × 换手率
```

简化公式（避免除零）：
```
MFV = (2 × 收盘价 - 最高价 - 最低价) / (最高价 - 最低价) × 换手率
```

- 一字板（最高价 = 最低价）时，MFV = 0
- 价格位置系数范围 [-1, 1]：收盘在最高点 = +1，收盘在最低点 = -1
- 使用换手率而非成交量/成交金额，天然消除不同规模指数的量纲差异

### 3.2 两种标准化方法

系统支持两种方法，通过配置切换：

#### 方法一：原始换手率（raw，默认）

```
AD_Line[t] = Σ_{i=t-64}^{t} MFV[i]    （65日滚动窗口）
```

直接累加65天的MFV，适合观察绝对资金活跃度。

#### 方法二：Z-score标准化（zscore）

```
基线: 过去250天（位于AD窗口之前，不重叠）
μ = 基线MFV均值, σ = 基线MFV标准差

对AD窗口内每天:
MFV_z[i] = (MFV[i] - μ) / σ

AD_Line_z[t] = Σ_{i=t-64}^{t} MFV_z[i]
```

Z-score反映的是"相对于自身历史的异常程度"，消除指数固有活跃度差异。  
基线窗口与AD窗口不重叠，避免信号稀释。  
数据不足315天或σ过低（< |μ| × 10%）时返回None，不参与排名。

### 3.3 机构吸筹/出货评级（A+ ~ E-）

在分类池内按 AD_Line 降序排列，计算百分位：

```
百分位 = 高于数 / (N-1) × 100
```

| 百分位区间 | 评级 | 含义 |
|-----------|------|------|
| ≥ 90% | A+ | 机构强烈吸筹（最佳） |
| 80% ≤ x < 90% | A | 机构强烈吸筹 |
| 70% ≤ x < 80% | B+ | 机构较强烈吸筹 |
| 60% ≤ x < 70% | B | 机构温和吸筹 |
| 50% ≤ x < 60% | C+ | 机构买入稍大于卖出，基本平衡 |
| 40% ≤ x < 50% | C | 机构买卖大致平衡 |
| 30% ≤ x < 40% | D+ | 机构温和出货 |
| 20% ≤ x < 30% | D | 机构大力出货 |
| 10% ≤ x < 20% | E | 机构强烈出货 |
| < 10% | E- | 机构强烈出货（最差） |

### 3.4 数字得分

```
AD_Score = (百分位 × 2) - 100     （范围 -100 ~ +100）
```

正分表示净吸筹，负分表示净出货。

## 4. 配置文件

`config/index_ad.yaml`：

```yaml
method: raw          # raw | zscore
window_days: 65      # 滚动窗口交易日数
ratings:             # 评级映射（只读参考）
  - { grade: "A+", min_pct: 90 }
  ...
```
