## 1. 产品概述

### 1.1 定位

对观察池 671 只股票回溯 3 个月的缠论买入信号表现，三组对比：
- **A**：缠论择时 vs 随机买入（同股票、同数量、不同日期）
- **B**：高/中/低置信度信号收益差异
- **C**：一买 vs 三买收益差异

### 1.2 核心价值

回答三个问题：
- 缠论信号是否比闭眼买更好？
- 背驰强度（置信度）和胜率正相关吗？
- 左侧抄底（一买）和右侧确认（三买）哪个更靠谱？

## 3. 数据来源

看板为**纯静态页面**，启动时从 `data.json` 加载数据，无需 API。

数据文件由 `chanlun_backtest_compare.py` 生成，结构：

```json
{
  "A_5d": {"label": "缠论 vs 随机", "chanlun_avg": 7.65, "chanlun_win": 100.0, "chanlun_n": 343, "random_avg": 1.69, "random_win": 53.2, "random_n": 662},
  "B_5d_高": {"avg": 8.05, "win": 100.0, "n": 151},
  "C_5d_一买": {"avg": 7.1, "win": 100.0, "n": 270},
  ...
}
```
