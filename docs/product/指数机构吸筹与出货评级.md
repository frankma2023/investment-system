# 产品说明文档：指数机构吸筹/出货评级（A/D Rating for Indices）

## 1. 概述

本产品将IBD（Investor's Business Daily）体系中经典的个股机构吸筹/出货评级（Accumulation/Distribution Rating）迁移至指数层面，用以衡量机构资金在某一指数（宽基、行业、主题、策略）上的近期净流入/流出程度。

该评级可以帮助投资者判断指数背后的机构资金动向，识别哪些指数正获得机构增持（吸筹），哪些正遭机构减持（出货），从而辅助自上而下的行业/板块配置决策。

## 2. 数据来源

| 数据项 | 来源 | 字段 | 频率 |
|--------|------|------|------|
| 指数日线行情 | index_daily_kline  | 开盘价、最高价、最低价、收盘价、成交量 | 每日 |
| 指数成交量 | index_daily_kline  | 成交量（股数） | 每日 |

> 需要至少13周（约65个交易日） 的历史数据来计算滚动评级。

## 3. 计算方法

### 3.1 日级别资金流量（Money Flow Value, MFV）

对每个指数、每个交易日，计算当日机构资金流入/流出的原始得分：

```
MFV = [(收盘价 - 最低价) - (最高价 - 收盘价)] / (最高价 - 最低价) × 成交量
```

**公式简化**（避免除零）：
- 若 `最高价 == 最低价`（一字板），则 MFV = 0。
- 否则：
  ```
  MFV = (2 × 收盘价 - 最高价 - 最低价) / (最高价 - 最低价) × 成交量
  ```

**取值范围**：
- 极端情况：收盘价在最高点时，`(2P - H - L)/(H-L) = 1`，MFV = +成交金额 → 极强流入。
- 极端情况：收盘价在最低点时，`(2P - H - L)/(H-L) = -1`，MFV = -成交金额 → 极强流出。
- 正常情况下，MFV 在 `(-成交金额, +成交金额)` 之间。

### 3.2 累积A/D线

对每个指数，计算过去13周（63-65个交易日） 的MFV累积和，得到累积资金线：

```
AD_Line[t] = Σ_{i=t-64}^{t} MFV[i]   （滚动窗口）
```

或者采用递推方式：
```
AD_Line[t] = AD_Line[t-1] + MFV[t]
```
每日更新，仅保留过去13周窗口（不截断亦可，但评级需要相对比较）。

### 3.3 机构吸筹/出货评级（A～E）

**关键**：A/D评级是一个**相对评级**，需要在同一时间点、同一指数分类池内进行横向比较。

**步骤**：
1. 选定比较池（如“全市场指数池29个”、“一级行业池11个”、“二级行业池45个”等）。
2. 对池内所有指数，获取当日最新的 `AD_Line` 值（过去13周累积值）。
3. 计算每个指数的 `AD_Line` 在池内的**百分位排名**（0～100%）。
4. 根据百分位映射为字母评级：

| 百分位区间 | 评级 | 含义 |
|-----------|------|------|
| ≥ 90% | A+ | 机构强烈吸筹（最佳） |
| 80% ≤ x < 90% | A | 机构强烈吸筹 |
| 70% ≤ x < 80% | B+ | 机构较强烈吸筹 |
| 60% ≤ x < 70% | B | 机构温和吸筹 |
| 50% ≤ x < 60% | C+ | 机构买入稍大于卖出，基本平衡 |
| 40% ≤ x < 50% | C | 机构买卖大致平衡 |
| 30% ≤ x < 40% | D+ | 机构温和出货 |
| 20% ≤ x < 30% | D | 机构大力出货 |
| 10% ≤ x < 20% | E | 机构强烈出货 |
| < 10% | E- | 机构强烈出货（最差） |

### 3.4 数字得分（可选）

同时输出一个**-100到+100**的连续分数，便于排序和计算：

```
AD_Score = (百分位 × 2) - 100
```

映射：百分位0 → -100，百分位50 → 0，百分位100 → +100。

正分表示净吸筹，负分表示净出货。
