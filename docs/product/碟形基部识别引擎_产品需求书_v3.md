# 碟形基部（Saucer Base）形态识别与突破检测 — 产品需求书 v3.0

> **突破日驱动 + 形态切割法 + 参数即标准**。判断"今天是否为碟形基部突破日"，是则输出信号。"好坏"由参数控制，不由评分判断。

---

## 1. 产品概述

本产品回答一个单一问题：**今天，某只股票是否出现碟形基部的有效突破信号？**

碟形基部是 William J. O'Neil 定义的七种经典基部形态之一。特征：回调浅（8%~20%）、时间长（≥7周）、弧线圆润、底部平坦、成交量极度萎缩。常见于大市值机构重仓股的牛市中途休整。

**核心流程**：检测日 t 的收盘价突破前高 + 0.01 → 向前回溯验证是否存在一个完整的碟形基部 → 满足所有条件则输出突破信号，不满足则不输出。

**信号是二元的**。没有"9 分的碟形"和"12 分的碟形"之分——只有"是碟形突破"和"不是碟形突破"。想要更严格的标准？收紧参数即可。

### 1.1 与其他基部形态的区别

| 维度 | 杯柄形态 | 双重底 | 扁平基部 | **碟形基部** |
|---|---|---|---|---|
| 回调深度 | 12%~33% | 15%~35% | 8%~15% | **8%~20%** |
| 底部形状 | U 形尖底 | W 形双底 | 水平窄箱体 | **浅圆弧形，底部平坦** |
| 形成时间 | 7~65 周 | 7~8 周以上 | 5~7 周 | **7~40 周** |
| 成交量 | 底部萎缩 | 右底量<左底量 | 全程持续萎缩 | **底部极度萎缩（< 50% 均量）** |
| 突破点 | 柄部上沿 | 中间峰最高价 | 基部最高点 | **碟口上沿（前高）** |
| 常见标的 | 中小盘成长股 | 中小盘 | 第二/三阶段基部 | **大盘股/机构重仓** |

## 2. 核心理念

### 2.1 碟形基部的本质

机构资金以"时间换筹码"完成建仓。盘子大，砸深了引发恐慌，拉急了吸引跟风盘推高成本。于是让股价像碟子一样慢慢滑落、在底部磨掉不坚定的持有者、再慢慢推回原位。

### 2.2 与圆弧底的区别

技术分析的"圆弧底"发生在长期下跌之后（熊市底部）。碟形基部发生在前置上涨之后（牛市中途）。外形相似，本质不同。前置上涨条件是关键分界线。

## 4. 量化计算规则

> **设计原则**：所有检测条件分两级——**必要条件**（不通过=不是碟形）和**可关开关**（默认开启，可按需关闭）。没有评分，只有参数。

### 4.1 分析周期策略

**周线主分析 + 日线精细化。**

碟形时间跨度长（2-6 个月），日线噪声多。周线识别结构，日线精确验证。

```
阶段一（周线）：周前复权收盘价 → 波峰波谷检测（窗口 13 周）
阶段二（日线）：日前复权收盘价 + 成交量 → 碟底平坦性、弧线圆润性、成交量确认
```

### 4.2 前置条件（全部必要条件）

```
条件 0.1: 前置上涨验证
          在前高确定后，在前高之前 120 个交易日内找最低点作为"前低"
          要求 (前高 - 前低) / 前低 ≥ 30%
          → 不满足？不是碟形基部（可能是熊市圆弧底）

条件 0.2: 趋势确认
          检测日 t 的收盘价 > SMA50

条件 0.3: 市值过滤（可关开关）
          流通市值 < min_market_cap_strict（默认 50 亿）→ 排除
          设 min_market_cap_strict = 0 可关闭此过滤
```

### 4.3 碟形基部识别（核心算法）

#### 步骤 1：定位碟底和前高（形态切割法）—— 必要条件

不靠固定时间窗口，靠价格行为自身的波动边界来切割形态。前高 = 从碟底逆向爬坡，爬到"前方出现 ≥ 20% 大波动"处。

```
1.1 定位碟底:

    在检测日 t 之前 ≥ 10 天、≤ 120 天范围内，找到最低周收盘价 = 碟底

    必要条件:
      - 碟底距 t ≥ 10 个交易日（回升已确认）
      - 碟底到 t 的涨幅 ≤ 20%（排除 V 形反转）

1.2 从碟底逆向爬坡找前高:

    初始化 candidate = 碟底。从碟底逐周向前回溯。

    对每一周:
      a) 如果本周收盘 > candidate → 更新 candidate = 本周收盘
      b) 如果本周是局部高点（前后各 13 周内最高）→ 对 candidate 执行三个切割:

         切割 A（上方切割）：
           candidate ≤ 检测日收盘 × (1 + cut_check_A_pct)  [默认 2%]
           → 不满足？candidate 太高，和今天不在同一形态周期，继续回溯

         切割 B（前方切割）：
           从 candidate 继续向前回溯，找第一个交易日 T 满足：
             |T收盘 - candidate| / min(T收盘, candidate) ≥ cut_pct  [默认 20%]
           找到 T → T 属于上一个形态，candidate = 前高，停止回溯
           回溯到底都没找到 → 取最早高点为前高

         切割 C（下方切割）：
           (candidate - 碟底) / candidate ∈ [correction_min, correction_max]
           默认 [8%, 20%]
           → 不满足？继续回溯

         三个切割全部通过 → candidate = 前高，停止回溯

1.3 兜底约束:

      - 前高距碟底 ≥ min_descent_weeks 周（默认 3）
      - 前高距碟底 ≤ 120 天（受碟底扫描范围约束，自然满足）
```

**三种切割的语义：**

| 切割 | 回答的问题 | 阈值来源 |
|---|---|---|
| A | 这个高点是否和今天突破同一形态周期？ | 突破日语义：前高不会明显高于突破日 |
| B | 这个高点前面是否有另一个独立形态？ | 碟形回调上限 20%：超此幅度已属另一形态 |
| C | 这个高点到碟底是否构成碟形回调？ | 碟形回调定义 [8%, 20%] |

#### 步骤 2：碟底平坦性验证（日线，双层窗口）—— 必要条件

碟形与杯柄的核心区别——碟底是平坦区间，不是尖底。

```
主检测（±5 天窗口）：
  - 碟底 ±5 天共 11 天，振幅 ≤ 5%
  - 线性回归斜率 |slope| ≤ 0.0005/天
  → 不通过？不是碟形

辅助验证（±10 天窗口）：
  - 碟底 ±10 天共 21 天，振幅 ≤ 8%
  → 不通过？输出时标记 bottom_quality = "shallow"
  → 可通过配置 bottom_flat_wide_required = true 将此升级为必要条件
```

#### 步骤 3：弧线圆润性验证（日线）—— 必要条件

碟形的下跌和上涨都应该是圆润的弧线。

```
条件 3.1: 将 [前高 → 碟底] 下行段按时间等分为三段:
          下行 A 段（前 1/3）→ 中段（中 1/3）→ 下行 B 段（后 1/3）

条件 3.2: 每段日均跌幅 ≤ segment_slope_max（默认 0.004/天）

条件 3.3: 三段递减: |slope_A| ≥ |slope_M| ≥ |slope_B|
          → 越近碟底越平缓

条件 3.4: 下行段日收益标准差 ≤ arc_smoothness_std（默认 0.008）
          → 波动均匀，没有大起大落

条件 3.5: 对称性: 上行段（碟底→t）日收益标准差 ≤ 下行段标准差 × 1.5
          → 可关开关（ascent_symmetry_enabled，默认 true）
          → 关闭后不检查对称性
```

#### 步骤 4：成交量验证（日线）—— 必要条件

```
条件 4.1: 碟底区间日均量 < 50 日均量 × vol_bottom_max（默认 0.50）

条件 4.2: 下行段后半日均量 < 下行段前半日均量 × vol_contraction（默认 0.60）
          → 越跌量越小

条件 4.3: 碟底区间内，单日量 > 50 日均量 × vol_spike_ratio 的天数 = 0（默认 1.2）
          → 可关开关（vol_spike_check，默认 true）
```

#### 步骤 5：突破信号检测 —— 必要条件

```
条件 5.1: 检测日收盘 ≥ 前高 + breakout_buffer（默认 0.01，即 A 股 1 分钱）
条件 5.2: 检测日成交量 ≥ 50 日均量 × breakout_vol_ratio（默认 1.3）
条件 5.3: 检测日为阳线（收盘 > 开盘）
条件 5.4: 收盘位置在当日区间上 50%（收盘在高位）
条件 5.5: 收盘 > SMA50 且 > SMA150（中期和长期上升趋势）
```

#### 步骤 6：虚假突破排除 —— 必要条件

以下任一情况出现则信号作废：
- 检测日或次日盘中跌破前高（以最低价计）
- 检测日成交量 < 50 日均量
- 检测日前 5 个交易日内存在放量下跌

### 4.4 柄部（Handle）检测 —— 可关开关

碟形可以带柄（Saucer With Handle），带柄成功率更高。默认开启，可关闭。

```
条件 H1: 在回升段定位柄部高点 — 回升段中最后一次收盘 ≥ 前高 × 0.90
条件 H2: 柄部低点距高点 ≥ 5 天 ≤ 30 天，回撤 ≤ handle_max_drawdown（默认 15%）
条件 H3: 柄部低点 ≥ 碟底 + 0.5 × (前高 - 碟底)，在上半部
条件 H4: 柄部日均量 < 基部整体日均量 × handle_vol_ratio（默认 0.70）
条件 H5: 柄部区间回归斜率 ≤ 0（向下或水平）
条件 H6: 柄部低点 ≥ SMA50

输出:
  - 有柄且全部通过 → pattern_type = "saucer_with_handle"
  - 无柄或柄部不通过 → pattern_type = "saucer"
  - 设 handle_required = true → 无柄时不输出信号（更严格）
```

## 5. 参数调优指南

没有评分卡。想调整信号的严格程度？直接改参数。

### 按"想达到什么效果"来调参

| 想达到的效果 | 改什么参数 | 怎么改 |
|---|---|---|
| 只抓教科书级碟形 | `correction_max` | 从 0.20 → 0.15（只接受 8%~15%） |
| 放宽回调深度 | `correction_max` | 从 0.20 → 0.25 |
| 容忍更深回调 | `correction_max` + `cut_pct` | 同步调高，保持一致性 |
| 成交量要求更严 | `vol_bottom_max` | 从 0.50 → 0.35 |
| 容忍略高成交量 | `vol_bottom_max` | 从 0.50 → 0.65 |
| 只抓带柄碟形 | `handle_required` | 从 false → true |
| 不需要柄部检测 | `handle_enabled` | 从 true → false |
| 碟底必须非常平 | `bottom_flat_wide_required` | 从 false → true |
| A 股波动大，放宽切割 | `cut_pct` | 从 0.20 → 0.25 |
| 排除小盘股 | `min_market_cap_strict` | 从 50 → 100 |
| 不过滤市值 | `min_market_cap_strict` | 设为 0 |

## 8. 回测与评估

| 指标 | 目标值 | 说明 |
|---|---|---|
| 信号准确率 | ≥ 55% | 突破后 20 日内涨幅 ≥ 5% |
| 平均收益 | ≥ 12% | 突破后 60 日平均最大收益 |
| 盈亏比 | ≥ 2.5:1 | 平均盈利 / 平均亏损 |
| 误报率 | ≤ 25% | 突破后 5 日内跌回碟口以下 |

**参数调优流程**：不是给信号打分，而是把参数作为旋钮——回测时跑参数网格，找准确率/误报率的最优交点，然后固定参数。不同的市场环境（牛市/熊市/震荡）可以有不同的参数组合。

**交叉验证**：同一股票同一天被多个形态识别时——碟形+杯柄→优先杯柄价格；碟形+扁平基部→标注"混合基部"；碟形+口袋支点→高价值信号。

## 10. FAQ 与 A 股适配

**Q1：碟形基部在 A 股出现频率如何？** 较杯柄少见。两阶段扫描——周线宽松筛选、日线严格确认。

**Q2：碟形基部 VS 平底形态？** 平底 ≤ 15% 横盘箱体，碟形 8%~20% 圆弧。

**Q3：碟形基部 VS 杯柄形态？** 碟底平坦性区分：碟形底平（振幅 < 5%），杯柄底尖。

**Q4：小盘股"碟形"？** < 50 亿排除（可调 `min_market_cap_strict`）。

**Q5：A 股波动大怎么调？** `cut_pct` 调到 25%，`correction_max` 调到 25%。

**Q6：怎么让信号更少但更准？** 收紧参数：`correction_max=0.15 vol_bottom_max=0.35 handle_required=true`。
