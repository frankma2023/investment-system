# 指数背离检测产品说明书 v2

> 状态：v2（投资专家挑战修正版），待审核  
> 变更：v1 → v2 量价平滑/RSI动态阈值/去掉综合评分/增加信号衰减

## 1. 产品概述

指数背离检测用于识别行情走势与技术指标之间的不一致，作为趋势衰竭的预警信号。背离本身不是买卖信号，需与其他模块（RS强度、拥挤度、A/D评级）综合使用。

**核心原则：** 四种背离独立展示，不做加权合并，用户交叉验证判断。

## 2. v1 背离类型

### 2.1 量价背离（优先级 P0）

**修正：** v1的点对点比较改为均线趋势比较，消噪。

```
价格MA5 = 5日收盘均价
成交量MA5 = 5日均量

价格趋势方向 = sign(今日MA5 - N日前MA5)
量能趋势方向 = sign(今日成交量MA5 - N日前成交量MA5)

顶背离：价格趋势方向 > 0 且 量能趋势方向 < 0（价涨量缩）
底背离：价格趋势方向 < 0 且 量能趋势方向 > 0（价跌量增）

强度 = |价格MA5变动%| + |成交量MA5变动%|
```

**参数（可配置）：**

| 参数 | 默认值 | 说明 |
|------|--------|------|
| MA周期 | 5日 | 价格和成交量的均线周期 |
| N | 10日 | 趋势对比回看天数 |
| 最低强度 | 3% | 强度低于此值不报（过滤噪音） |
| 有效期 | 5日 | 背离信号最长保留天数 |

### 2.2 RSI背离（优先级 P0）

**修正：** 超买超卖线改为动态百分位，自适应指数波动特征。

```
RSI(14) = 100 - 100 / (1 + 平均涨幅14日 / 平均跌幅14日)

超买线 = 该指数过去250日RSI的90%分位值
超卖线 = 该指数过去250日RSI的10%分位值

顶背离：
  - 当前价格 > 过去N日高点
  - 当前RSI < 过去N日RSI峰值
  - 当前RSI > 超买线

底背离：
  - 当前价格 < 过去N日低点
  - 当前RSI > 过去N日RSI谷值
  - 当前RSI < 超卖线
```

**参数：**

| 参数 | 默认值 | 说明 |
|------|--------|------|
| RSI周期 | 14日 | |
| N | 20日 | 新高/新低回看窗口 |
| 超买分位 | 90% | 250日RSI分位 |
| 超卖分位 | 10% | 250日RSI分位 |
| 有效期 | 5日 | |

### 2.3 MACD背离（优先级 P0）

```
EMA12 = 12日指数移动平均
EMA26 = 26日指数移动平均
MACD柱 = (EMA12 - EMA26) - 9日EMA(MACD)

顶背离：
  - 当前价格 > 过去N日高点
  - 当前MACD柱 < 过去N日MACD柱峰值
  - MACD柱 > 0（零轴上方）

底背离：
  - 当前价格 < 过去N日低点
  - 当前MACD柱 > 过去N日MACD柱谷值
  - MACD柱 < 0（零轴下方）
```

**参数：**

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 快线/慢线/信号线 | 12/26/9 | 标准MACD参数 |
| N | 20日 | 新高/新低回看窗口 |
| 有效期 | 5日 | |

### 2.4 成分股上涨比例背离（优先级 P1）

```
上涨比例 = 当日上涨成分股数 / 成分股总数

底背离（跌多涨少但改善中）：
  - 今日指数涨
  - 今日上涨比例 < 昨日（参与度下降）
  - 今日上涨比例 < 50%
  - 连续2日满足

底背离：
  - 今日指数跌
  - 今日上涨比例 > 昨日（参与度改善）  
  - 今日上涨比例仍 < 40%
  - 连续2日满足
```

**参数：**

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 连续天数 | 2日 | 连续满足才计为背离 |
| 顶背离阈值 | 50% | |
| 底背离阈值 | 40% | |
| 有效期 | 3日 | |

**v1限制：** 仅计算 market/sector_l1/sector_l2 池（86个指数），thematic/strategy 暂不计算。

## 3. 信号等级（四种背离统一）

| 等级 | 条件 |
|------|------|
| ⚠️ 潜在 | 背离首次出现，价格尚未确认 |
| 🔴 确认 | 顶背离+跌破近5日低点 或 底背离+突破近5日高点 |
| 🔴🔴 强烈 | 确认条件满足 + 均线交叉确认（5日线穿10日线）|

## 4. 信号衰减与清除

背离信号最长保留5日（成分股背离3日）。超期未确认的自动清除，下次触发重新计算。已确认的信号保留至反向信号出现或价格突破关键均线。

## 5. 与前序模块联动

| 场景 | 解读 |
|------|------|
| RS强度A+ 且 顶背离 | 最强指数可能见顶，警惕 |
| A/D评级E 且 底背离 | 出货动力衰竭，可能反转 |
| 拥挤度"高拥挤" 且 顶背离 | 拥挤+背离，回调风险最高 |
| 三者同时出现 | 多条独立信号共振，最值得关注 |

**联动实现：** 在背离看板中增加三列（RS强度评级、A/D评级、拥挤度），直接展示该指数在已有模块中的最新评级，用户一目了然。

## 6. API设计

```
GET /api/index-divergence?pool=market&date=2026-05-06
```

响应中每个指数返回四种背离的独立状态，不做加权合并：

## 7. 回测看板

### 左侧
- 指数池 + 日期选择
- 卡片1-4：四种背离的参数配置（可折叠）
- 「计算」+「保存配置」

### 右侧
- 统计卡片：总指数 / 顶背离 / 底背离 / 强烈背离 / 无背离
- 背离汇总表：代码、名称、量价、RSI、MACD、成分股、RS强度、A/D评级、拥挤度
- 背离列用颜色标记（红=顶、绿=底、深浅=等级），无背离为空
- 点击代码弹窗成分股（复用）


## 9. 与v1的对比

| 变更项 | v1 | v2 |
|--------|----|----|
| 量价背离 | 单日点对点 | 5日均线趋势对比 |
| RSI超买超卖 | 固定70/30 | 动态百分位(90%/10%) |
| 综合评分 | 加权合成单一分数 | 取消，四列独立 |
| 信号管理 | 无衰减 | 5日有效期自动清除 |
| 前序联动 | 无 | 表中展示RS/A/拥挤度评级 |
