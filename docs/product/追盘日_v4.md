# 追盘日（Follow-Through Day）

## 一、核心意义

追盘日是指市场在经历一轮下跌后，从反弹尝试起点算起的第 4～10 天之间，出现的一根放量大阳线。它标志着机构资金开始系统性买入，确认下跌趋势结束，新一轮上升趋势开始。

> 欧奈尔取的是反弹尝试后的第 4 ~ 7 天，我自己修改为 4 ~ 10 天。

## 二、反弹尝试日（DAY 1）

### 定义

简单来说，反弹尝试的第一天（Day 1），必须是市场下跌过程中，出现"潜在支撑信号"的那一天。

> 在追盘日规则中，反弹尝试当日为 Day 1，次日为 Day 2，依此类推 

它需要同时满足两个核心条件：价格创新低 + 收盘表现出强势。

创阶段新低：
- 当日最低价 < 最近20个交易日最低价

收盘强势：（以下二者取一）
- 收盘上涨：当日收盘价 > 前一日收盘价
- 当日收盘价 > (当日最低价 + 当日最高价) x 0.5

> 要求严格创出新低，即当前最低价低于过去20个交易日（不含当前）的最低点。
> 收盘强势最小涨幅阈值 > 0.01%

### 反弹尝试失败重置

反弹尝试可能失败，如果失败则重置计数器，重新开始寻找新的反弹尝试。

反弹尝试失败有2个可能：
- 1. 在同一个反弹尝试窗口（第2-10天）内，如果出现更低的低点且收盘强势，应视为新的 Day1，重置计数器；
- 2. 如果同一个反弹尝试窗口（第4-10天）内没有出现追盘日，导致反弹尝试失败，重置计数器。

## 三、追盘日

追盘日（Follow-Through Day, FTD）是在一次反弹尝试窗口内（从 Day1 算起的第4至第10天），确认市场可能进入上升趋势的关键信号

### 3.1 量化规则

| 序号  | 条件        | 量化标准                                                |
| --- | --------- | --------------------------------------------------- |
| 1   | 时间窗口  | 该交易日必须是所在反弹尝试窗口内的 第 4 ~ 10 天（以 Day1 为第1天）。 |
| 2   | 价格涨幅  | 当日收盘价比前一日收盘价 上涨超过 1%（即涨幅 > 1.0%）。               |
| 3   | 成交量放大 | 当日成交量 大于前一日成交量（即放量）。                            |
| 4   | 结构完整 | 从 Day1 到该交易日（含）之间，每一天的最低价 都没有跌破 Day1 的最低价。      |

### 3.2 异常处理

- 追盘日当天不能同时是抛盘日
- 连续追盘日去重（相邻两天都触发取第一天）
- 如果同一天同时满足反弹尝试日和追盘日条件，优先判定为追盘日。

### 3.3 追盘日失效的条件

> 在真正的大牛市来临之前，你可能会看到5到7个虚假的追盘日信号。
> 追盘日本质上是一个概率信号，而不是确定性信号。它告诉你"市场目前具备开启一轮上升趋势的条件"，但无法保证"这次一定会成功"。

| 条件 | 量化标准  |
| :------- | :--------------------------------------------- | 
| 破位失效 | 追盘日后5个交易日内，指数收盘价跌破追盘日的最低价 |
| 无力上涨 | 追盘日后10个交易日内，指数从追盘日收盘价起的最大涨幅 < 2% |
| 抛盘覆盖 | 追盘日后20个交易日内，抛盘日数量 ≥ 3个 |
| 回撤过深 | 追盘日后15个交易日内，指数从最高点回撤幅度超过追盘日以来最大涨幅的 50%  |


### 3.4 追盘成功确认的条件

满足以下任一条件即可认为该追盘日已被成功确认，趋势有效确立。

| 条件   | 量化标准   | 
| :-------- | :------- | 
| 涨幅确认 | 从追盘日收盘价起，未来10个交易日内，指数最大涨幅 ≥ 4%  | 
| 均线确认 | 追盘日后15个交易日内，5日均线上穿10日均线，且10日均线上穿20日均线  |
