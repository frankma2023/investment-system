
## 三、大盘环境判断卖出规则

### 3.1 为什么大盘环境决定卖出时机

欧奈尔的研究表明：**75%的个股走势与大盘方向一致**。这意味着，即使你持有一只基本面优异、技术形态完美的股票，如果大盘进入下跌趋势，它也极大概率会跟随下跌。因此，**在大盘出现明确的顶部信号或转入下降趋势时，应主动减仓或清仓**，而不是等到个股破位才行动。

大盘环境判断的核心目标：
- **规避系统性风险**：避免在熊市中持股被套。
- **保护利润**：在牛市顶部及时锁定收益。
- **提高资金效率**：等待下一轮上升趋势再重仓出击。

### 3.2 判断大盘顶部的量化信号

欧奈尔体系通过追踪主要股指（如沪深300、中证全指、创业板指）的**价格行为、成交量、市场宽度**等指标来识别顶部。

#### 3.2.1 抛盘日（Distribution Days）累积

**定义**：抛盘日是指指数下跌且成交量放大的交易日，反映机构派发。当一段时间内抛盘日数量达到阈值，表明市场上升动能衰竭。`distribution_days_detail`数据表已经有数据。

**量化规则**：
| 抛盘日数量（过去25个交易日） | 信号强度 | 操作建议 |
|------------------------------|----------|----------|
| ≤3 个 | 正常 | 维持仓位 |
| 4 个 | 轻度预警 | 减仓 25% → 评分卡 +10 分 |
| 5 个 | 中度预警 | 减仓 50% → 评分卡 +20 分 |
| ≥6 个 | 强烈卖出信号 | 减仓 ≥75% → 评分卡 +40 分 |

**高位定义**：指数收盘价 ≥ 52 周最高价 × 0.95（即距 52 周高点不足 5%）。
**高位权重加倍**：若指数处于高位，抛盘日最终得分 ×2（4个→20分，5个→40分，≥6个→60分）。工程实现时也可将高位期间的下跌日判定条件放宽，使抛盘日计数增加。

**抛盘日判定标准**（任一满足即计入）：
- 价格条件：当日收盘价较前一日下跌 ≥ 0.2%（可配置）。
- 成交量条件：当日成交量 ≥ 前一日成交量（或 ≥ 10日均量 × 1.1）。
- 若指数处于高位（距52周高点<5%），抛盘日权重加倍。

#### 3.2.2 追盘日失效

**定义**：追盘日是欧奈尔判断下降趋势转为上升趋势的确认信号。如果在上升趋势中，之前确认的追盘日信号失效（例如随后出现大量抛盘日或指数跌破追盘日低点），则表明上升趋势可能已经结束。

**量化规则**：
- 查找过去 30 个交易日内是否存在一个已确认的追盘日（反弹第 4-7 天出现放量上涨，`is_valid=1`）。
- 若无追盘日，此项不触发（0 分）。
- 若存在追盘日且满足以下任一条件，视为"追盘日失效"：
  - 随后 10 个交易日内出现 ≥3 个抛盘日。
  - 指数收盘价跌破追盘日 K 线的最低点。
  - 数据库中 `invalidated_date` 不为空（已被历史标记为失效）。
- 触发后评分卡 +30 分。

#### 3.2.3 龙头股先见顶

**定义**：市场中的领涨龙头股（通常是前期涨幅最大、RS评级最高的股票）往往比大盘指数提前几天甚至几周见顶。密切跟踪这些股票的走势，可以提前预警。

**量化规则**：
- **龙头股池**：每周一定池，筛选全市场 RS(250) 百分位 ≥ 99 的所有股票（通常 30-50 只），记录入池时各自的收盘价作为基准价。池子一周内不变，下周一重新筛选。
- **回落标准**：相较各自入池基准价，累计跌幅 > 10%。
- **触发条件**：池中 ≥ 50% 的股票触发回落标准 → 龙头股见顶信号成立，评分卡 +25 分。
- 辅判信号（不单独计分，用于增强置信度）：池内股票出现放量下跌、跌破 50 日均线、高点逐步降低等。

#### 3.2.4 垃圾股补涨

**定义**：在牛市的末期，低价股、ST股、问题股等基本面差的股票开始疯狂补涨，而前期龙头股开始滞涨或下跌。这是典型的“鸡犬升天”阶段，是牛市即将结束的强烈信号。

**量化规则**：
- **分组**：每日按全市场个股股价升序排列，取后 20% 作为低价股池，前 20% 作为高价股池。
- **计算**：分别计算低价股池和高价股池的等权平均 20 日涨幅。
- **触发条件**：低价股池涨幅 > 高价股池涨幅 × 1.5，且该状态持续 ≥ 5 个交易日。
- 出现该信号后，应大幅减仓。

#### 3.2.5 指数与成分股背离

**定义**：指数上涨，但大多数个股下跌（或指数下跌但多数个股上涨）。这种背离表明指数被权重股扭曲，市场真实宽度恶化。

**量化规则**：
- **覆盖指数**：中证全指、上证指数、深证成指、创业板指、科创50，分别计算各自**成分股**的涨跌比。
- **领涨指数**：每周一定池，从 `index_rs_daily` 取 5 个指数的最新 `rs_60`，降序第一名为当周领涨指数。
- **背离判定**（仅监测顶部背离）：指数日涨幅 > 0.5%，但其成分股中上涨家数占比 < 40%。
- **计分规则**：任一指数连续 2 天背离 → 20 分；领涨指数背离 → 权重 ×2（即 40 分）。多个指数同时背离分别计分，但背离信号累计上限 60 分。

### 3.3 跌破关键均线的卖出规则

> **基准指数**：均线系统统一使用**中证全指（000985）**计算。中证全指覆盖全市场，比沪深300更具代表性。下文所有"指数"均指中证全指。

#### 3.3.1 跌破50日均线（10周线）

**定义**：50日均线是中期趋势的生命线。当中证全指收盘价跌破50日均线且无法快速收复，表明中期趋势可能已经转弱。

**量化规则**：
- **第一次跌破**：收盘价低于50日均线。若次日或3日内收复，可暂观察；若5日内无法站回，减仓50%。
- **确认跌破**：收盘价低于50日均线，且50日均线开始拐头向下（今日MA50 < 5日前MA50），清仓。

#### 3.3.2 跌破200日均线（40周线）

**定义**：200日均线是牛熊分界线。当中证全指收盘价跌破200日均线，表明长期趋势可能已经逆转，应彻底离场。

**量化规则**：
- 收盘价低于200日均线，且200日均线开始拐头向下 → 无条件清仓。
- 若只是短暂跌破（1-2天）并迅速收复，可保留部分仓位，但需提高警惕。

#### 3.3.3 均线死叉

**定义**：中证全指的50日均线下穿200日均线（死亡交叉），是长期趋势转熊的确认信号。

**量化规则**：
- 出现50日均线向下穿越200日均线（即MA50 < MA200且之前MA50 > MA200）→ 清仓。


### 3.4 市场宽度与成交量指标

#### 3.4.1 上涨/下跌家数比

**定义**：每日全市场上涨家数与下跌家数的比值，反映市场参与广度。

**量化规则**：
- 计算AD比率 = 上涨家数 / 下跌家数。
- 若AD比率连续5个交易日 < 0.7（即下跌家数比上涨家数多30%以上），表明市场宽度恶化，减仓50%。
- 若AD比率 < 0.5，清仓。

#### 3.4.2 新高/新低家数

**定义**：创52周新高股票数量与创52周新低股票数量的对比。

**量化规则**：
- 计算NH/NL比率 = 创新高家数 / 创新低家数。
- **数据来源**：`market_health.py` 的 `compute_hl_ratio()` 实时计算 52 周 N日新高新低比，取 5 日均值平滑。
- NH/NL 比率 < 0.5 且持续 3 天 → 评分卡 +30 分，触发熔断"清仓级"信号。

#### 3.4.3 成交量趋势

**定义**：指数上涨过程中，成交量是否逐步萎缩；指数下跌过程中，成交量是否放大。

**量化规则**：
- 上升趋势中，若严格连续 5 日指数上涨（每日涨幅 > 0）但日均成交量 < 20 日均量 × 80%，表明上涨无量 → 评分卡 +20 分。
- 下跌趋势中，若出现连续3日放量下跌（每日成交量 > 50日均量 × 1.3），表明恐慌盘涌出，清仓。

### 3.5 综合判断流程与仓位管理

#### 3.5.1 大盘卖出信号评分卡

| 信号 | 条件 | 分数 |
|------|------|------|
| 抛盘日（25日） | 4个 | 10分 |
| 抛盘日（25日） | 5个 | 20分 |
| 抛盘日（25日） | ≥6个 | 40分 |
| 抛盘日·高位 | 距52周高点<5%，权重×2 | ×2 |
| 追盘日失效 | 30日内追盘日确认后被跌破 | 30分 |
| 龙头股见顶 | >50%龙头股回落>10% | 25分 |
| 垃圾股补涨 | 低价股涨幅>高价股1.5倍 | 25分 |
| 指数背离 | 5个主要指数任一连续2天成分股背离 | 20分 |
| 指数背离（领涨） | 领涨指数连续2天背离（权重×2） | 40分 |
| 跌破50日线 | 收盘跌破且5日未收复 | 30分 |
| 跌破200日线 | 收盘跌破 | 80分 |
| 均线死叉 | MA50下穿MA200 | 60分 |
| AD比率 | <0.7连续5天 | 20分 |
| AD比率 | <0.5 | 60分 |
| NH/NL比率 | <0.5连续3天 | 30分 |
| 上涨无量 | 连续5日涨但量<20日均量×80% | 20分 |

**决策**：
- 总分 < 60：正常仓位（70%-100%）
- 60 ≤ 总分 < 100：减仓至 50%
- 100 ≤ 总分 < 140：减仓至 25%
- 总分 ≥ 140：清仓

**熔断规则（与评分并行，触发即生效，无需总分达标）**：
- 单信号触发"清仓级"（AD<0.5 / 跌破200日线 / 均线死叉）：仓位上限 ≤ 25%，且必须在3个交易日内完成减仓。
- 任意两个"清仓级"信号同时触发：**强制清仓**（仓位 0%），次日开盘即执行，不允许保留任何持仓。

#### 3.5.2 执行流程

```
每日收盘后执行：
1. 计算主要股指（沪深300、中证全指、创业板指）的抛盘日数量。
2. 检查追盘日是否仍有效。
3. 监控龙头股列表（Top 10 RS股票）是否出现卖出信号。
4. 计算全市场宽度指标（AD比率、NH/NL比率）。
5. 检查指数是否跌破关键均线。
6. 根据评分卡计算总分，决定次日仓位。
7. 若总分 ≥ 140，无条件清仓所有个股，转入现金或货币基金。
```

### 3.7 总结

| 核心要点 | 说明 |
|----------|------|
| **大盘优先** | 75%个股跟随大盘，判断大盘顶部是卖出决策的第一步 |
| **量化信号** | 抛盘日、追盘日失效、龙头股见顶、垃圾股补涨、均线破位等均可量化 |
| **评分体系** | 将多个信号加权评分，根据总分决定仓位 |
| **严格执行** | 大盘卖出信号一旦触发，应无条件减仓/清仓，不抱侥幸心理 |

大盘环境判断卖出规则是欧奈尔体系中保护资本的最后一道防线。它不以个股基本面为转移，而是基于市场整体风险收益比。当市场环境恶化时，最好的策略就是持有现金，等待下一轮上升趋势。
