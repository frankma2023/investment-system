1. **保留两套模式**：快速模式（点对点，短期）+ 慢速模式（均线趋势，中期），用户可自选。
2. **成分股背离优化**：宽基/一级行业用全成分股，二级/主题用前20大权重股。

# 指数背离检测产品说明书 v3

> 状态：v3（根据专家挑战修正版），待开发  
> 变更：v2 → v3 增加双模式/成分股权重优化/确认窗口延长/联动共振/灵敏度滑块/筛选器

## 1. 产品概述

指数背离检测用于识别行情走势与技术指标之间的不一致，作为趋势衰竭的**预警信号**。背离本身不是买卖信号，需与其他模块（RS强度、拥挤度、A/D评级）综合使用。

**核心原则：** 四种背离独立展示，不做加权合并，用户交叉验证判断。支持用户自选**灵敏度**（短线/长线）及**快速/慢速量价模式**。

## 2. v3 背离类型

### 2.1 量价背离（优先级 P0）

**提供两种模式供用户选择：**

| 模式 | 计算方式 | 适用场景 |
|------|----------|----------|
| 快速模式（点对点） | 今日收盘价 vs 昨日收盘价；今日成交量 vs 昨日成交量 | 捕捉短期价格异动，适合短线预警 |
| 慢速模式（均线趋势） | 价格MA5 vs 10日前MA5；成交量MA5 vs 10日前MA5 | 过滤噪音，适合中长线趋势 |

**共用逻辑：**
- 顶背离：价格向上且量能向下（价涨量缩）
- 底背离：价格向下且量能向上（价跌量增）
- 强度 = |价格变动%| + |成交量变动%|（快速模式用单日，慢速用均线变化）
- 强度低于阈值（默认3%）时不报警

**参数（可配置）：**

| 参数 | 快速模式默认 | 慢速模式默认 | 说明 |
|------|-------------|-------------|------|
| MA周期 | 1（不使用） | 5日 | 慢速模式使用的均线周期 |
| 对比回看N | 1日 | 10日 | 趋势方向对比天数 |
| 最低强度 | 3% | 3% | 低于此值不报 |
| 有效期 | 10日 | 10日 | 信号最长保留 |

### 2.2 RSI背离（优先级 P0）

**RSI超买超卖线支持固定阈值与动态分位二选一（用户可配置）：**

| 模式 | 超买线 | 超卖线 | 说明 |
|------|--------|--------|------|
| 固定阈值 | 70 | 30 | 传统经典，对极端行情可能钝化 |
| 动态分位 | 过去250日RSI的90%分位 | 过去250日RSI的10%分位 | 自适应波动，但牛市可能抬升 |

**顶背离条件：**
- 当前价格 > 过去N日高点
- 当前RSI < 过去N日RSI峰值
- 当前RSI > 超买线（所选模式）

**底背离条件：**
- 当前价格 < 过去N日低点
- 当前RSI > 过去N日RSI谷值
- 当前RSI < 超卖线（所选模式）

**参数：**

| 参数 | 短线默认 | 长线默认 | 说明 |
|------|----------|----------|------|
| RSI周期 | 7日 | 14日 | 周期越短越敏感 |
| N（新高/新低窗口） | 10日 | 20日 | |
| 超买/超卖线模式 | 动态分位 | 固定阈值 | 用户任选 |
| 有效期 | 10日 | 10日 | |

### 2.3 MACD背离（优先级 P0）

标准MACD参数（12/26/9），顶背离要求MACD柱>0，底背离要求MACD柱<0。

**参数：**

| 参数 | 短线默认 | 长线默认 |
|------|----------|----------|
| 快线/慢线/信号线 | 12/26/9（不变） | 12/26/9 |
| N（新高/新低窗口） | 10日 | 20日 |
| 有效期 | 10日 | 10日 |

### 2.4 成分股上涨比例背离（优先级 P1）

**计算范围：**
- 市场指数（29个）、一级行业（11个）、二级行业（45个） → 使用**全成分股**
- 主题指数（229个）、策略指数（93个） → 使用**前20大权重股**（按自由流通市值排序）

**顶背离（价涨参与度降）：**
- 今日指数上涨
- 今日上涨比例 < 昨日上涨比例
- 今日上涨比例 < 50%
- 连续2日满足

**底背离（价跌参与度升）：**
- 今日指数下跌
- 今日上涨比例 > 昨日上涨比例
- 今日上涨比例 < 40%
- 连续2日满足

**参数：**

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 连续天数 | 2日 | |
| 顶背离阈值 | 50% | |
| 底背离阈值 | 40% | |
| 有效期 | 5日 | |

## 3. 信号等级（四种背离统一）

| 等级 | 条件 |
|------|------|
| ⚠️ 潜在 | 背离首次出现，价格尚未确认 |
| 🔴 确认 | 顶背离 + 跌破近**20日**低点；或 底背离 + 突破近**20日**高点 |
| 🔴🔴 强烈 | 确认条件满足 + 均线交叉确认（5日线上穿10日线 for 底背离；5日线下穿10日线 for 顶背离） |

## 4. 信号衰减与自动延展

- **有效期**：默认10日（成分股背离5日）。超期未确认则自动清除。
- **自动延展**：若信号已触发但尚未确认，且价格继续向背离方向运动（如顶背离后继续上涨），则有效期延长5日。最多延展一次。

## 5. 与前序模块联动（增强版）

### 5.1 看板增加列
- RS强度评级（A/B/C/D/E）
- A/D评级（A/B/C/D/E）
- 拥挤度等级（低/正常/偏高/高）

### 5.2 综合预警文字
系统自动生成一行文字，例如：
```
“RS强度A+，拥挤度高，顶背离 → 强烈回调风险”
“RS强度C，A/D评级E，底背离 → 可能反转”
```

### 5.3 共振强度等级（用户可勾选参与共振的模块）
用户勾选RS强度、A/D评级、拥挤度中的若干项，系统根据组合给出共振强度：

| 共振强度 | 条件 |
|----------|------|
| 高 | 勾选模块中≥2个发出极端信号（如RS强度A+、拥挤度高、A/D评级E）且有背离 |
| 中 | 1个极端信号 + 其他正常 |
| 低 | 无极端信号，仅背离 |

## 6. 用户灵敏度选择

在回测看板顶部增加**灵敏度滑块**，预设两种模式，也可自定义参数：

| 模式 | RSI周期 | N窗口 | 量价模式 | 确认窗口 | 有效期 |
|------|---------|-------|----------|----------|--------|
| 短线（敏感） | 7日 | 10日 | 快速（点对点） | 10日低点/高点 | 10日 |
| 长线（稳健） | 14日 | 20日 | 慢速（均线） | 20日低点/高点 | 10日+延展 |

用户切换模式时，所有相关参数自动更新。

## 7. 回测看板功能

### 7.1 左侧面板
- 指数池选择（market/sector_L1/sector_L2/thematic/strategy）
- 日期选择
- 灵敏度选择（短线/长线/自定义）
- 量价模式切换（快速/慢速）
- RSI超买超卖线模式选择（固定/动态）
- 参与共振的模块勾选（RS强度/A/D评级/拥挤度）
- 四个背离的参数折叠配置
- 多条件筛选器（见7.3）

### 7.2 右侧汇总区域
- **统计卡片**：总指数 / 顶背离 / 底背离 / 强烈背离 / 无背离
- **背离汇总表**（列）：
  - 代码、名称、收盘价、涨跌幅
  - 量价背离（类型+等级）、RSI背离、MACD背离、成分股背离
  - RS强度评级、A/D评级、拥挤度等级
  - 综合预警文字、共振强度等级
- 背离列用颜色标记：红色=顶背离，绿色=底背离，深浅=等级（潜在/确认/强烈）

### 7.3 多条件筛选器
支持组合筛选：
- 背离类型（顶/底/无）
- RS评级（A/B/C/D/E）
- A/D评级（A/B/C/D/E）
- 拥挤度等级（低/正常/偏高/高）
- 共振强度等级（高/中/低）

筛选后表格只显示满足所有条件的指数。

## 8. 技术实现要点

### 8.1 计算频率
- **每日盘后计算**（约16:00后），无盘中实时推送。
- 信号确认依赖当日收盘价，无误导。

### 8.2 数据依赖
- 理杏仁 API 提供指数日线、成分股日线、自由流通市值。
- 成分股背离中，主题/策略指数使用前20大权重股（按自由流通市值），需每日获取权重数据。

### 8.3 性能优化
- 成分股背离仅对86个（市场+一级+二级）计算全成分股，其他用前20大，大幅降低计算量。
- 所有背离结果预计算存入 `index_divergence_daily` 表，回测看板直接读取。

## 9. 已知局限与未来迭代

| 局限 | 计划迭代 |
|------|----------|
| 未回测验证背离有效性 | 上线运行3个月后，根据实际数据出具回测报告 |
| RSI动态分位在极端行情可能失效 | 后续增加“混合模式”（取固定与动态的较低者作为超买线） |
| 成分股背离暂不覆盖所有主题指数 | 后续可增加按成交额占比筛选主题指数前20大权重股 |
