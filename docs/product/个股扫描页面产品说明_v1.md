# 个股扫描页面产品说明 v1.0

> 最后更新：2026-05-10  
> 页面路径：`web/stock-valuation/`  
> 数据来源：`fundamental_indicator` + `stock_financials_annual` + `stock_equity_change` + `/api/stock-analysis`

## 1. 产品定位

从**估值、盈利、健康、强度、评估**五个维度对单只股票进行全维度快速体检。输入股票代码即可一键查看该股的所有关键指标，辅助选股和持仓监控。

## 2. 页面布局

顶部搜索框 + 年份选择 + 五 Tab 切换。

### 2.1 搜索选择器

- **股票代码输入框**：6 位代码，默认 600519（贵州茅台）
- **年份范围**：起始 2016~2020，截止 ~2026

### 2.2 Tab 切换

| Tab | 内容 | 图表数 |
|------|------|:--:|
| 📊 估值仪表 | PE-TTM/PB/PS-TTM/股息率/总市值走势 | 5 |
| 💰 盈利质量 | ROE/EPS/毛利率/营收增速/净利增速/自由现金流 | 6 |
| 🏥 财务健康 | 资产负债率/有息负债率/流动比率/速动比率/应收周转/存货周转 | 6 |
| 📈 市场强度 | RPS走势等（待开发） | — |
| 🧮 财务评估 | DCF估值/可比公司/盈利趋势/三表预测（移植自 stock-analysis） | 4 卡片 |

## 3. 各 Tab 详细说明

### 3.1 估值仪表

| 图表 | 数据来源 | 说明 |
|------|---------|------|
| PE-TTM | `fundamental_indicator.pe_ttm` | 市盈率走势，含历史均值线 |
| PB | `fundamental_indicator.pb` | 市净率走势 |
| PS-TTM | `fundamental_indicator.ps_ttm` | 市销率，适合成长股评估 |
| 股息率 | `fundamental_indicator.dyr` | 百分比折线 |
| 总市值 | `fundamental_indicator.mc` | 亿元折线 |

每张图下方含分析文字：当前值、历史均值、最高、最低。

### 3.2 盈利质量

柱状图展示近十年年报数据，每图含十年均值虚线：

| 图表 | 数据来源 | 单位 |
|------|---------|:--:|
| ROE | `stock_financials_annual.roe` | % |
| EPS | 净利润 ÷ 总股本（`stock_equity_change`） | 元 |
| 毛利率 | `stock_financials_annual.gross_margin` | % |
| 营收增速 | `stock_financials_annual.revenue_yoy` | % |
| 净利增速 | `stock_financials_annual.net_profit_yoy` | % |
| 自由现金流 | `stock_financials_annual.free_cash_flow` | 亿 |

### 3.3 财务健康

| 图表 | 数据来源 | 说明 |
|------|---------|------|
| 资产负债率 | `stock_financials_annual.asset_liability_ratio` | 总负债/总资产 |
| 有息负债率 | `stock_financials_annual.interest_bearing_debt_ratio` | 需付息的债务占比 |
| 流动比率 | `stock_financials_annual.current_ratio` | 流动资产/流动负债 |
| 速动比率 | `stock_financials_annual.quick_ratio` | 剔除存货后的流动比率 |
| 应收周转率 | `stock_financials_annual.receivables_turnover` | 应收账款回收效率 |
| 存货周转率 | `stock_financials_annual.inventory_turnover` | 库存管理效率 |

### 3.4 市场强度

待开发。规划展示：RPS_20/60/120/250 走势、价格 vs MA50/MA200、RS 线、成交量趋势。

### 3.5 财务评估

完整移植 `stock-analysis` 页面功能，包含四个评估卡片（2 列网格）：

| 卡片 | 核心指标 | 详解功能 |
|------|---------|:--:|
| 💰 DCF估值 | 当前价 / 目标价 / 企业价值 / WACC / 年度预测表 / 敏感性 | ✅ |
| 📊 可比公司分析 | 中位PE/PB/PS / 可比公司对比表 | ✅ |
| 📈 盈利趋势 | 营收同比 / 净利同比 / 毛利率 / ROE / 季度明细表 | ✅ |
| 📋 三表预测 | 基准年参数 / 三年营收/毛利/净利/经营CF/自由CF预测表 | ✅ |

每张卡右上角「📖 详解」按钮在卡片下方展开对应模块的完整推演分析（含参数溯源、计算步骤、使用建议等长文解读）。

## 4. 技术实现

### 4.1 数据流

**估值数据**：
```
GET /api/stock-valuation?code=600519&start=2016-01-01&end=2026-05-07
  → 查询 fundamental_indicator (pe_ttm/pb/ps_ttm/dyr/mc)
  → ECharts 折线图
```

**财务数据**：
```
GET /api/stock-financials?code=600519
  → 查询 stock_financials_annual + stock_equity_change
  → EPS = 净利润 / 总股本
  → ECharts 柱状图
```

**财务评估**：
```
GET /api/stock-analysis?code=600519
  → DCF/Comps/Earnings/3-Statement 四项分析
  → 卡片 + 详解推演面板
```

### 4.2 文件结构

```
web/stock-valuation/
└── index.html          ← 单文件页面

src/server.py           ← /api/stock-valuation /api/stock-financials 端点
```

### 4.3 样式

小红书手账风格：`theme.css` + `base.css` + `xhs-cards.css` + `components.css`。图表使用 ECharts 5.5。

## 5. 使用场景

- 选股前快速体检：从估值、盈利、健康三个维度扫一眼
- 持仓监控：定期查看持仓股的 ROE 趋势是否下滑、负债率是否攀升
- 估值验证：DCF 目标价 vs 当前价，判断低估/高估
- 同行业对比：切换同行业不同股票，对比毛利率、ROE、负债结构
