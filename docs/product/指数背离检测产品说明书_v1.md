# 指数背离检测产品说明书 v1

> 状态：v1 草案，待审核  
> 数据源：index_daily_kline + daily_kline + index_constituents  
> 参考：docs/product/指数背离的初步想法.md

## 1. 产品概述

指数背离检测用于识别行情走势与技术指标之间的不一致，作为趋势衰竭的预警信号。背离本身不是买卖信号，需与其他模块（RS强度、拥挤度、A/D评级）综合使用。

v1 聚焦四种可立即落地的背离类型，按数据可得性和信号可靠性排序。

## 2. v1 背离类型

### 2.1 量价背离（优先级 P0）

**算法：** 比较近期价格趋势与成交量趋势的方向一致性。

```
价格趋势 = (今日收盘 / N日前收盘 - 1) × 100
成交量趋势 = (今日成交量 / N日前成交量 - 1) × 100

背离判定：
- 顶背离：价格趋势 > 阈值 且 成交量趋势 < -阈值
- 底背离：价格趋势 < -阈值 且 成交量趋势 > 阈值
```

**参数（可配置，默认值）：**

| 参数 | 默认值 | 说明 |
|------|--------|------|
| N | 20日 | 趋势回看天数 |
| 趋势阈值 | 2% | 价格/成交量变动幅度门槛 |

**信号等级：**
| 等级 | 条件 |
|------|------|
| 潜在 | 出现1次背离 |
| 确认 | 连续2日背离，或N日内出现2次 |
| 强烈 | 连续3日背离，且价格趋势绝对值>5% |

### 2.2 RSI背离（优先级 P0）

**算法：** 比较价格走势与RSI走势的背离。

```
RSI(14) = 100 - 100 / (1 + 平均涨幅14日 / 平均跌幅14日)

顶背离判定（价格新高，RSI未能新高）：
  - 当前价格 > N日前高点 且 当前RSI < N日前RSI_peak
  - 且 当前RSI > 超买线（默认70）

底背离判定（价格新低，RSI未能新低）：
  - 当前价格 < N日前低点 且 当前RSI > N日前RSI_trough  
  - 且 当前RSI < 超卖线（默认30）
```

**参数（可配置）：**

| 参数 | 默认值 | 说明 |
|------|--------|------|
| RSI周期 | 14日 | RSI计算周期 |
| N | 20日 | 新高/新低回看窗口 |
| 超买线 | 70 | 顶背离需RSI在此之上 |
| 超卖线 | 30 | 底背离需RSI在此之下 |

**信号等级：**
| 等级 | 条件 |
|------|------|
| 潜在 | 出现1次背离 |
| 确认 | RSI已从超买/超卖区回落/回升 |
| 强烈 | 价格和RSI均突破关键位置（跌破/站上均线） |

### 2.3 MACD背离（优先级 P0）

**算法：** 经典MACD指标背离检测。

```
EMA12 = 12日指数移动平均
EMA26 = 26日指数移动平均
MACD = EMA12 - EMA26
Signal = 9日EMA(MACD)
Histogram = MACD - Signal

顶背离（价格新高，MACD柱线峰值走低）：
  - 当前价格 > N日前高点
  - 当前MACD柱线 < N日前MACD柱线峰值
  - 且 MACD柱线 > 0（零轴上方）

底背离（价格新低，MACD柱线谷值走高）：
  - 当前价格 < N日前低点
  - 当前MACD柱线 > N日前MACD柱线谷值
  - 且 MACD柱线 < 0（零轴下方）
```

**参数：**

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 快线 | 12 | EMA快线周期 |
| 慢线 | 26 | EMA慢线周期 |
| 信号线 | 9 | Signal线周期 |
| N | 20日 | 新高/新低回看窗口 |

**信号等级：**
| 等级 | 条件 |
|------|------|
| 潜在 | 出现背离 |
| 确认 | MACD柱线与Signal线发生交叉确认 |
| 强烈 | DIFF线（MACD线）下穿/上穿零轴 |

### 2.4 成分股上涨比例背离（优先级 P1）

**算法：** 指数价格上涨，但成分股中上涨家数占比反而下降。

```
上涨比例 = 当日上涨成分股数 / 成分股总数

背离判定（顶背离）：
  - 今日指数收盘 > 昨日收盘
  - 今日上涨比例 < 昨日上涨比例
  - 今日上涨比例 < 50%（跌多涨少）
  - 连续N日满足以上条件

背离判定（底背离）：
  - 今日指数收盘 < 昨日收盘  
  - 今日上涨比例 > 昨日上涨比例
  - 今日上涨比例仍 < 40%（整体偏弱但改善）
  - 连续N日满足以上条件
```

**参数：**

| 参数 | 默认值 | 说明 |
|------|--------|------|
| N | 3日 | 连续背离天数 |
| 顶背离阈值 | 50% | 上涨比例低于此值才计为背离 |
| 底背离阈值 | 40% | 上涨比例低于此值但改善中 |

**注意：** 此类型需要遍历成分股计算涨跌，对229个主题指数可能较慢。v1仅对market/sector_l1/sector_l2启用（共86个指数），thematic/strategy延后。

## 3. 综合背离评分

每个指数每日计算四种背离类型，汇总为综合评分：

```
综合评分 = 量价背离权重×2 + RSI背离权重×2 + MACD背离权重×2 + 成分股背离权重×1

每类背离：潜在=1分，确认=2分，强烈=3分，无背离=0分
顶背离为正分（警告），底背离为负分（机会）

综合等级：
  0分: 无背离
  1-3分: 轻度背离
  4-6分: 中度背离
  7分以上: 强烈背离
```

## 4. 背离确认过滤

根据概念文档要求，背离只是预警信号，需价格确认才生效：

| 背离方向 | 确认条件 |
|----------|---------|
| 顶背离 | 收盘价跌破近5日低点，或5日均线下穿10日均线 |
| 底背离 | 收盘价突破近5日高点，或5日均线上穿10日均线 |

未确认的背离标记为"潜在"，已确认的标记为"有效"。

## 6. 回测看板布局

小红书风格，左侧参数配置 + 右侧结果展示。

### 左侧
- 指数池选择 + 日期选择
- 卡片1：量价背离参数（N日、趋势阈值）
- 卡片2：RSI背离参数（周期、超买超卖线、N日）
- 卡片3：MACD背离参数（快线/慢线/信号线、N日）
- 卡片4：成分股背离参数（连续天数、阈值）
- 卡片5：综合评分权重
- 「计算」+「保存配置」按钮

### 右侧
- 统计卡片：总指数数 / 顶背离数 / 底背离数 / 强烈背离数
- 背离汇总表：代码、名称、最新价、量价背离、RSI背离、MACD背离、成分股背离、综合评分、综合等级
- 每列背离用颜色标记：红=顶背离、绿=底背离、深浅=信号等级
- 点击指数代码弹出成分股弹窗（复用现有组件）

## 7. v1/v2 分工

| 背离类型 | v1 | 说明 |
|----------|:--:|------|
| 量价背离 | ✅ | 数据最充足，算法最简单 |
| RSI背离 | ✅ | 经典指标，可靠性高 |
| MACD背离 | ✅ | 经典指标，与RSI互补 |
| 成分股上涨比例背离 | ✅ | 仅覆盖86个指数 |
| 指数间RS背离 | 🔜 | 需定义"最强指数"选择逻辑 |
| NH-NL背离 | 🔜 | 需全市场个股新高新低计算 |
| 多周期共振 | 🔜 | 日线+周线联合判断 |
| 成交额占比背离 | 🔜 | 需行业板块成交额分布数据 |
