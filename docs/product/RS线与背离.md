# RS线（相对强度线）完全指南

RS线（Relative Strength Line）是欧奈尔体系中仅次于RS评级的第二重要的个股分析工具。它与RS评级互补——**RS评级解决“是否强”的问题，RS线解决“是否持续变强”的问题**。

---

## 一、RS线的定义与计算

### 1.1 基本公式

最简单的RS线公式如下：

```
RS线 = 个股前复权收盘价 / 基准指数收盘价
```

这个比值衡量的是：**每1单位的指数涨幅，对应多少单位的个股涨幅**。

举例：
- 中证全指 = 5000 点
- 某股票价格 = 50 元
- 此时 RS 线 = 50 / 5000 = 0.01

### 1.2 为什么原始值“不直观”？

由于数值差距悬殊（个股几十元，指数几千点），RS线的绝对值通常在 0.001～0.1 之间，不适合直接阅读。因此需要**归一化**：

```
RS线归一化值_t = (RS线_t / RS线_基日) × 100
```

**基日选择建议**：
- **个股视角**：使用个股上市首日（或数据起始日）作为基日，可以看到从上市至今的相对强度变化
- **策略视角**：使用策略上线首日作为基日（适合做回测看板）

归一化后的RS线从 **100** 开始。
- > 100：个股跑赢大盘
- < 100：个股跑输大盘
- 持续上升：相对强度增强
- 持续下降：相对强度减弱

### 1.3 基准指数选择

| 基准类型 | 适用场景 | 示例 |
| :--- | :--- | :--- |
| **全市场基准** | 判断个股相对于整体市场的强弱 | 中证全指（000985） |
| **行业基准** | 判断个股相对于所属行业的强弱 | 申万一级/二级行业指数 |
| **风格基准** | 判断个股相对于特定风格（如成长/价值）的强弱 | 创业板指 / 科创50 |

**欧奈尔原教旨**：使用标普500作为基准。在A股的实践中，建议主用 **中证全指** 作为全市场基准，同时可附加 **行业基准** 做精细化分析。

## 二、RS线的核心逻辑：为什么它能判断强弱？

RS线的本质是**个股与大盘的赛跑**。我们可以把它分解成两部分来看：

### 2.1 数学分解

将上式取对数后得到：
```
ln(RS线) = ln(个股价格) - ln(指数价格)
```
然后对时间求导：
```
d/dt ln(RS线) = (个股涨幅率) - (指数涨幅率)
```

核心结论：**RS线的斜率 = 个股收益率 - 指数收益率**。
- RS 线斜率 > 0：个股比大盘涨得快（或跌得慢）
- RS 线斜率 < 0：个股比大盘涨得慢（或跌得快）

这是为什么 RS 线可以直接衡量“个股是否跑赢大盘”的理论基础——它比较的不是价格高低的绝对差异，而是两者在**时间维度上的变化速率差**。

### 2.2 理解RS线的三个层次：一段完整的趋势观察

RS线的趋势可以通过三个典型阶段直观地理解：

- **上升**：个股比大盘强。大盘涨时，个股涨更多；大盘跌时，个股跌更少。
- **走平**：个股与大盘同步。两者涨跌幅度相近，没有明显超额收益。
- **下降**：个股比大盘弱。大盘涨时跟涨乏力；大盘跌时，个股跌更多。

**欧奈尔名言的量化表达**：“你应该只买入那些RS线正在上升的股票，而不是那些RS线正在下降的股票。”——这句话的本质是：不要试图去接下跌的飞刀，不要买入那些正在跑输大盘的弱势股。

## 三、RS线背离：股价与RS线打架

**背离**指股价走势与RS线走势出现方向不一致。产生背离的根本原因是：股价的上涨可能由大盘带动（β收益），而非个股自身的阿尔法驱动。RS线能够剥离大盘的影响，反映**个股自身的真实强度**。

### 3.1 顶背离（卖出信号）

**表现形式**：
- 股价创出新高（或接近新高）
- RS线未能创出新高，甚至走平或下降

**为什么会背离？**
> 假设个股从 50 元涨到 60 元，按理很强。但如果同期大盘从 5000 点涨到 6000 点，个股相对强度实际在下降——它的涨幅被大盘的牛市掩盖了。这被量化为 RS 线从 0.01 下降到 0.01（120/年区间内未超过历史峰值）。

**量化规则**：
```
顶背离条件:
1. 今日股价 > 过去N日（如20日）最高价
2. 今日RS线 < 过去N日RS线最高值
3. 今日RS线 < 昨日RS线（或RS线斜率转负）
```

**操作含义**：这是**最危险的卖出信号之一**。股价创新高，但内在强度不足，通常是“最后的拉升”或“假突破”，机构在掩护出货。

### 3.2 底背离（买入信号）

**表现形式**：
- 股价创出新低（或接近新低）
- RS线未创新低，甚至抬高

**为什么会背离？**
> 假设个股从 60 元跌到 45 元，看似很弱。但如果同期大盘从 6000 点暴跌到 4500 点，个股的跌幅相对更小，RS 线反而从 0.01 上升到了 0.011，说明它在熊市中表现出了抗跌的韧性。一旦大盘企稳，它可能率先反弹。

**量化规则**：
```
底背离条件:
1. 今日股价 < 过去N日（如20日）最低价
2. 今日RS线 > 过去N日RS线最低值
3. 今日RS线 > 昨日RS线（或RS线斜率转正）
```

**操作含义**：个股正在悄悄变强，只是被大盘的熊市掩盖。一旦大盘企稳，这类股票往往最先反弹，是重点关注的候选标的。

### 3.3 背离的确认与过滤

单日背离信号可能有噪音，建议增加确认机制：

| 确认方法 | 规则 | 说明 |
| :--- | :--- | :--- |
| **连续背离** | 连续N日（如3日）满足背离条件 | 可过滤短期波动造成的假信号 |
| **强度过滤** | RS线峰值/谷值的偏离幅度 ≥ 一定阈值 | 要求RS线与股价的差异足够显著，避免微小波动导致误判 |
| **时间范围** | N日窗口可配置，短线用10日，长线用50日 | 不同交易周期对应不同的确认窗口 |
| **成交量确认** | 顶背离时要求上涨缩量（或下跌放量） | 若股价创新高但缩量，同时RS线背离，信号可靠性提升 |

## 四、RS线的实战应用框架

| 市场环境 | RS线状态 | 操作策略 |
| :--- | :--- | :--- |
| **牛市/上升期** | RS线持续上升 | 坚定持有，这是最赚钱的阶段 |
| **牛市/上升期** | RS线开始走平或下降 | 换股，将资金转移到RS线仍在上升的更强标的 |
| **熊市/下跌期** | RS线持续下降 | 离场观望，不要试图抄底（个股正在加速跑输市场） |
| **熊市/下跌期** | RS线走平或上升 | 重点关注，这是下跌市中的抗跌品种，大盘企稳后可能率先反弹 |

## 五、RS线与RS评级的分工

| 工具 | 关注的问题 | 计算方式 | 更新频率 | 核心用途 |
| :--- | :--- | :--- | :--- | :--- |
| **RS评级** | 个股在过去一年中，在全市场中的排位是多少？ | 横截面百分位排名 | 每日 | 筛选强势股，建立候选池 |
| **RS线** | 个股相对于大盘的强度趋势是上升还是下降？ | 时间序列比值：个股/指数 | 每日（图表/趋势判断） | 判断买卖时机，识别顶/底背离 |

**组合使用**：
1. 用 RS 评级（≥ 80）筛选出强势候选股。
2. 用 RS 线确认这些股票正在跑赢大盘（RS线向上）或出现底背离（RS线抬高）。
3. 验证成功后，进入具体的买点分析（如杯柄形态、口袋支点等）。

## 六、代码实现（Python版）

```python
import pandas as pd
import numpy as np

def calculate_rs_line(stock_prices, index_prices, base_date=None):
    """
    计算RS线及归一化值
    
    参数:
        stock_prices: Series, 个股前复权收盘价，索引为日期
        index_prices: Series, 基准指数收盘价，索引为日期
        base_date: 基日（用于归一化），默认为数据起始日
    """
    # 合并数据
    merged = pd.DataFrame({'stock': stock_prices, 'index': index_prices}).dropna()
    
    # 计算原始RS线
    merged['rs_line_raw'] = merged['stock'] / merged['index']
    
    # 归一化：基日值 = 100
    if base_date is None:
        base_date = merged.index[0]
    base_value = merged.loc[base_date, 'rs_line_raw']
    merged['rs_line_norm'] = (merged['rs_line_raw'] / base_value) * 100
    
    # 计算RS线趋势（20日均线）
    merged['rs_line_ma20'] = merged['rs_line_norm'].rolling(20).mean()
    
    return merged[['rs_line_raw', 'rs_line_norm', 'rs_line_ma20']]

def detect_divergence(price_series, rs_line_series, lookback=20, threshold=0.02):
    """
    检测股价与RS线的背离
    
    返回:
        'top_divergence' / 'bottom_divergence' / None
    """
    # 近期高点和低点
    price_high = price_series.rolling(lookback).max()
    price_low = price_series.rolling(lookback).min()
    rs_high = rs_line_series.rolling(lookback).max()
    rs_low = rs_line_series.rolling(lookback).min()
    
    # 顶背离：股价新高但RS线未新高
    top_divergence = (price_series == price_high) & (rs_line_series < rs_high.shift(1))
    
    # 底背离：股价新低但RS线未新低
    bottom_divergence = (price_series == price_low) & (rs_line_series > rs_low.shift(1))
    
    return top_divergence, bottom_divergence
```

## 七、总结

**一句话**：RS 线是个股的“照妖镜”——它能剥掉大盘的伪装，看清个股的真实强度。

| 你的需求 | RS线提供的答案 |
| :--- | :--- |
| 这只股票真的强吗？ | RS线是否持续上升 |
| 股价新高，能追吗？ | RS线是否同步新高（是则跟，否则警惕） |
| 股价新低，能抄底吗？ | RS线是否提前走强（是则关注，否则观望） |
| 持仓股该持有还是卖出？ | RS线是否还在创出新的高点（走平或下降应考虑退出） |
| 选股池里哪只最强？ | RS线的累计上升幅度最大（跑赢大盘最多） |

RS线不是金股筛选器，而是**趋势跟踪和风险管理工具**。它不直接告诉你“买哪只”，而是帮您判断“持有哪些、放弃哪些”。
