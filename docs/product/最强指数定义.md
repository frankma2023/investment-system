# 最强指数

## 一、核心概念

**最强指数**对应的是L（领军股）维度——不仅要选最强的个股，也要选最强的板块/指数。板块效应是筛选牛股最核心的过滤器之一。历史数据显示，股价上涨动力中，50%来自行业趋势，37%来自大盘趋势，只有12%来自个股本身。


## 二、A股指数层级体系

在A股市场中，指数层级可以按以下维度划分，具体指数池参见`config/index_style.yaml`：

### 2.1 层级一：全市场指数池（10只）

覆盖全市场或大板块的综合性指数。

目标：筛选出本指数池符合以下条件的最强3只指数，以RS_20降序排列。
若指数数量不足3只，只显示实际符合条件的指数。

- RS_250 ≥ 80
- RS_60 ≥ 85
- RS_20 ≥ 90
- MA50 > MA150 > MA200
- 过去20日A/D线（累积/派发线）斜率 > 0

### 2.2 层级二：一级行业指数池（11个）

覆盖全市场的中证一级行业指数。

若指数数量不足5只，只显示实际符合条件的指数。
