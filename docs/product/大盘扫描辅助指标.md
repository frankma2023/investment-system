> **抛盘日/追盘日** 是欧奈尔判断市场状态的**核心主线信号**;
> 以下指标属于**辅助确认指标**,用来**过滤掉虚假信号**或**提高仓位调整的信心**。

下面按照"可直接量化实现"的标准,给出每个指标的定义、量化规则和计算步骤。

---

## 1. 涨跌家数比(Advance/Decline Ratio)

**定义**
当日上涨个股数 ÷ 下跌个股数,反映市场整体情绪是普涨还是普跌。

**量化规则**
- 正常牛市:比值 > 1.5
- 健康震荡:0.8 ~ 1.5
- 普跌/弱势:< 0.6
- 极端背离:指数上涨但比值 < 0.6 → 机构拉权重出货

**计算步骤**(A股全市场)
1. 统计当日 **上涨个股数**(收盘价 > 前收盘)
2. 统计当日 **下跌个股数**(收盘价 < 前收盘)
   *平盘不计入*
3. 比值 = 上涨数 / 下跌数

> 推荐使用 **5日均值平滑**,避免单日噪声。

---

## 2. 新高新低比(新高/新低比)

**定义**
创52周新高个股数 ÷ 创52周新低个股数。
欧奈尔认为:健康牛市应 **新高数 >> 新低数**。

**量化规则**
- 牛市健康区:比值 ≥ 2.0
- 危险信号:比值 < 1.0(新低超过新高)
- 强烈看跌:比值 < 0.5(需暂停选股)

**计算步骤**
1. 统计全市场 **过去52周(约252个交易日)**:
   `52周新高` = 当日最高价 ≥ 过去252日最高价
   `52周新低` = 当日最低价 ≤ 过去252日最低价
2. 比值 = 新高数 / 新低数
3. 推荐使用 **5日均值** 或 **10日均值**

> ⚠️ 上市不满252天的个股:使用其可用的全部历史数据。

---

## 3. MA50上方占比(% of Stocks Above 50-day MA)

**定义**
收盘价在50日均线上方的个股占全市场比例。
这是欧奈尔体系中最核心的宽度指标之一。

**量化规则**
- 强势市场:≥ 70%
- 健康调整:40% ~ 70%
- 弱势/熊市:< 40%
- 超卖极端:< 20%(常伴随反弹)

**计算步骤**
1. 对每只个股计算 **50日简单移动平均**(需要至少50个交易日数据)
2. 判断:`close > MA50`
3. 占比 = 满足条件的个股数 / 全市场有效个股数 × 100%
4. 输出 **5日均值** 或直接使用原始值

---

## 4. 放量突破数(Volume Breakout Count)

**定义**
当日放量突破关键形态/价格位的个股数量。
这是"机构活动"的直接体现。

**量化规则**
- 简化定义:`volume > MA50(volume) × 1.5` 且 `close > 前20日最高价`
> 标准定义:放量 + 突破杯柄/基底高点(但形态难以统一识别,建议使用简化定义)

**计算步骤(简化版,适合量化)**
1. 对每只个股计算:
   - `突破条件A`:收盘价 > 过去20日最高价
   - `放量条件B`:成交量 > MA50(成交量) × 1.5
2. 同时满足 A + B 计为 **1个放量突破**
3. 累加全市场数量
4. 观察趋势(高于过去20日均值1.5倍 → 活跃)

> 该指标更适用于**每日扫描预警**,而非固定阈值判断。

## 5. 板块轮动（Sector Rotation）

**定义**  
领涨板块的变化趋势，判断资金是流入强势板块还是分散/撤退。

**数据源**  
使用 `config/index_style.yaml` 中的 4 个行业指数池：

| 池 | 数量 | 用途 | 指标 |
|------|:--:|------|------|
| `sector_l1` | 11 | 参与健康评分 | Top 5 重叠率 |
| `sector_l2` | 45 | 参与健康评分 | Top 5 重叠率 |
| `theme` | 229 | 仅统计展示 | Spearman 秩相关系数 |
| `strategy` | 93 | 仅统计展示 | Spearman 秩相关系数 |

各池独立计算，不跨池比较。

**计算步骤（L1/L2 小池 — Top 5 重叠率）**  
1. 对池内每个指数，从 `index_daily_kline` 取最近收盘价  
2. 计算 **5日收益率** = (今日收盘价 − 5日前收盘价) / 5日前收盘价  
3. 按收益率降序排名，取 Top 5  
4. 对比上周同期 Top 5 列表，统计重叠个数  
5. 健康分 = (L1重叠得分 + L2重叠得分) / 2

量化规则：  
- 重叠 ≥ 4 个 → 10 分（领涨板块稳定，趋势健康）  
- 重叠 = 3 个 → 8 分  
- 重叠 = 2 个 → 6 分  
- 重叠 = 1 个 → 4 分  
- 重叠 = 0 个 → 0 分（全盘洗牌，风险信号）

**计算步骤（Theme/Strategy 大池 — Spearman 秩相关系数）**  
1. 对池内每个指数计算 5 日收益率  
2. 生成本周排名列表和上周排名列表  
3. 计算 Spearman 秩相关系数 ρ：
   ```
   ρ = 1 - (6 × Σd²) / (n × (n² - 1))
   ```
   其中 d = 每个指数本周排名 − 上周排名，n = 池内指数总数  
4. 结果区间 [-1, 1]：
   - ρ > 0.7 → 资金持续抱团，轮动健康  
   - ρ 0.3 ~ 0.7 → 正常轮动  
   - ρ < 0.3 → 全榜大洗牌，资金无序乱窜，谨慎

> 大池的秩相关系数仅在大盘扫描看板中展示，不参与健康评分卡打分。小池因数量少更适合 Top 5 重叠率，大池因覆盖面广更适合秩相关衡量全榜稳定性。

## 6. 恐慌/贪婪指数(Fear & Greed Index)

**定义**
综合波动率、期权、资金流向等维度的情绪合成指标。
欧奈尔本人不直接使用该指标,但可用于辅助识别极端情绪。

**量化规则(简化版,适合A股)**
综合3个维度:
- 波动率维度:ATR(20) / 收盘价(越高越恐慌)
- 宽度维度:MA50上方占比(越低越恐慌)
- 动量维度:过去5日涨跌幅(负越多越恐慌)

**计算步骤(极简版)**
1. 对每个子指标计算 **历史百分位(过去252日)**
2. 恐慌程度 = 平均值
   - > 80% → 极度贪婪
   - < 20% → 极度恐慌
3. 极度恐慌 ≥ 80% → 潜在反弹前兆(与追盘日配合使用)

> 不建议作为主线信号,仅用于"极端区域"提示。

## 7. 融资余额变化(Margin Balance Change)

**定义**
沪深两市融资余额的短期变化,反映杠杆资金态度。

**量化规则**
- 融资余额 **5日连续增加** → 杠杆资金看多
- 融资余额 **单日下降 > 1%** → 杠杆资金撤退信号
- 与指数走势背离:指数涨但融资余额降 → 谨慎

**计算步骤**
1. 获取每日 **两市融资余额合计**
2. 计算:`当日变化率 = (今日余额 - 昨日余额) / 昨日余额 × 100%`
3. 计算 **5日累计变化率**
4. 规则:
   - 5日累计 > 1.5% → 杠杆积极
   - 5日累计 < -1.5% → 杠杆撤退

> 该指标存在**滞后性**(收盘后才发布),不适合盘中决策,适合日度复盘。
