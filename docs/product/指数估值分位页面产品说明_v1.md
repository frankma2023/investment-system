# 指数估值分位页面产品说明 v1.0

> 最后更新：2026-05-09  
> 数据来源：`index_fundamental_daily` 表（理杏仁 /index/fundamental API）  
> 页面路径：`web/index-valuation/`

## 1. 产品定位

为用户提供任意指数的 **PE-TTM / PB / 股息率** 历史走势及 **十年分位** 可视化查询。帮助判断当前估值在历史中的相对位置——贵了还是便宜了，辅助择时和仓位决策。

## 2. 页面布局

顶部导航栏 + 搜索选择器 + 三张时序图表。

### 2.1 导航栏

与其他页面统一：🏠 看板 | 回测 ▾ | 指数扫描 | 估值分位（高亮）| 个股财务 | 大盘扫描 | 仓位管理

### 2.2 搜索选择器

- **搜索框**：输入代码或名称实时筛选下拉列表
- **下拉选择器**：408 个指数的 `代码 + 名称`
- **起始年份**：2016 / 2018 / 2020 / 2022 / 2024
- **截止年份**：2023~至今

### 2.3 统计卡片

6 格卡片展示选中指数当前值：

| 卡片 | 数据 |
|------|------|
| PE-TTM | 当前 PE 数值 |
| PE 十年分位 | 百分比 + 颜色标识（红=高估/黄=偏高/绿=低估） |
| PB | 当前 PB 数值 |
| PB 十年分位 | 百分比 + 颜色标识 |
| 股息率 | 当前股息率 |
| 股息率十年分位 | 百分比 + 颜色标识 |

### 2.4 图表区

三张 ECharts 图表，上下排列：

| 图表 | 内容 | 对比基准 |
|------|------|---------|
| PE-TTM 走势 | PE 值折线 + 十年分位% 虚线 | 沪深300 PE 灰色虚线 |
| PB 走势 | PB 值折线 + 十年分位% 虚线 | 沪深300 PB 灰色虚线 |
| 股息率走势 | 股息率% 折线 + 十年分位% 虚线 | 沪深300 股息率灰色虚线 |

**图表特性**：

- 滚轮缩放 + 底部滑块拖拽（dataZoom inside + slider）
- 图例可点击切换显示/隐藏各条线
- 分位图 70% 和 30% 处各有参考虚线标注「偏高」「偏低」
- 颜色：PE=红、PB=蓝、股息率=绿、分位=琥珀色、沪深300=灰色

### 2.5 分析文字

PE 和 PB 图表下方各有一段约 150 字的自动分析，包含：

- 当前数值 + 十年分位百分比 + 高低判断（严重高估/偏高/中性/偏低/严重低估）
- 近十年均值、最高值、最低值
- 与沪深300 的横向对比
- 操作建议

## 3. 技术实现

### 3.1 数据流

```
选择指数 + 年份范围 → GET /api/valuation?code=xxx&start=...&end=...
                     → 查询 index_fundamental_daily 表
                     → 返回 {dates, pe, pe_pct, pb, pb_pct, dyr, dyr_pct}
                     → ECharts 渲染折线图 + 生成分析文本
```

### 3.2 API 端点

`GET /api/valuation?code=000300&start=2016-01-01&end=2026-05-07`

返回格式：
```json
{
  "code": "000300",
  "dates": ["2016-01-04", ...],
  "pe": [13.5, ...],
  "pe_pct": [0.45, ...],
  "pb": [1.62, ...],
  "pb_pct": [0.38, ...],
  "dyr": [0.025, ...],
  "dyr_pct": [0.55, ...]
}
```

### 3.3 十年分位说明

十年分位 = 当前估值在过去 10 年所有交易日中的百分位。由理杏仁 API 原生计算（`pe_ttm.y10.mcw.cvpos`），非本地计算。取值 0~1，1=历史最高位，0=历史最低位。

### 3.4 文件结构

```
web/index-valuation/
└── index.html          ← 单文件页面

src/server.py           ← /api/valuation 端点
```

### 3.5 样式

小红书手账风格：`theme.css` + `base.css` + `xhs-cards.css` + `components.css`。图表使用 ECharts 5.5 本地副本。

## 4. 使用场景

- 选指数前快速看一眼估值水位，避免在高位接盘
- 对比不同指数的分位，找估值洼地（如全指消费 PE 分位 85% vs 全指医药 PE 分位 30%）
- 季度复盘时查看主要持仓指数的估值变化趋势
- 结合拥挤度指标：高拥挤+高估值=双杀信号
- 结合 RS 强度：低估值+RS 拐头=潜在买点
