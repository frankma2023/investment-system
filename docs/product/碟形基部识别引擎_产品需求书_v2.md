# 碟形基部（Saucer Base）形态识别与突破检测 — 产品需求书 v2.0

> **融合版本**：保留周线主分析、柄部独立检测、0-10 分评分卡等实战设计；注入碟底驱动算法、弧线圆润性量化、前置上涨验证、碟底平坦性检测等精确规则。

---

## 1. 产品概述

本产品用于自动识别股票 K 线图中的**碟形基部（Saucer Base）** 形态，并在价格放量突破时发出买入信号。

碟形基部是 William J. O'Neil 定义的七种经典基部形态之一，特征为**回调浅、时间长、弧线圆润、成交量极度萎缩**。它常见于大市值机构重仓股的牛市中途休整——机构以"时间换筹码"的方式悄悄完成吸筹，整个过程缓慢而温和。

**核心目标**：量化识别碟形基部结构，捕捉有效的突破买点，为 CAN SLIM 选股提供技术面验证。

**在整体系统中的位置**：个股扫描模块 > 基部识别引擎的子模块。与杯柄、双重底、扁平基部并列，共享 K 线数据管道和回测看板。

### 1.1 与其他基部形态的区别

| 维度 | 杯柄形态 | 双重底 | 扁平基部 | **碟形基部** |
|---|---|---|---|---|
| 回调深度 | 12%~33% | 15%~35% | 8%~15% | **8%~20%（浅但有弧度）** |
| 底部形状 | U 形尖底 | W 形双底 | 水平窄箱体 | **浅圆弧形，底部平坦** |
| 形成时间 | 7~65 周 | 7~8 周以上 | 5~7 周 | **7~40 周（通常较长）** |
| 成交量 | 底部萎缩 | 右底量<左底量 | 全程持续萎缩 | **底部极度萎缩（< 50% 均量）** |
| 突破点 | 柄部上沿 | 中间峰最高价 | 基部最高点 | **碟口上沿（左侧高点）** |
| 常见标的 | 中小盘成长股 | 中小盘 | 第二/三阶段基部 | **大盘股/机构重仓** |

## 2. 核心理念

### 2.1 碟形基部的本质

碟形基部是**机构资金以"时间换筹码"的方式完成建仓**。因为标的盘子大，如果砸得太深会引发恐慌抛售（机构自己也出不来），如果拉得太急会吸引跟风盘推高成本。所以机构选择最温和的方式——让股价像碟子一样慢慢滑落，在底部横盘磨掉不坚定的持有者，再慢慢推回原位。

**"最慢的形态，最稳的突破"** 是碟形基部的核心特征。

### 2.2 与圆弧底（Rounding Bottom）的区别

技术分析中的"圆弧底"通常发生在长期下跌之后（熊市底部），而碟形基部发生在**前置上涨之后**（牛市中途）。两者在外形上相似，但本质不同：

- 圆弧底 = 熊市末期的筑底反转
- 碟形基部 = 牛市中途的休整再出发

**务必验证前置上涨条件**（见 §4.1），否则会将长期熊股误判为碟形基部。

## 3. 形态结构定义

```
左侧高点（碟口上沿）────────────────── 突破买点
   \                                   /
    \    ┌─────────────────────┐      /
     \   │  成交量极度萎缩      │     /
      \  │  碟底平坦区间        │    /
       \ │  （振幅 < 5%）       │   /
        \│                     │  /
         └─────────────────────┘
          ←── 碟底区间（±5天）──→
```

### 3.1 关键特征（按重要性排序）

| 优先级 | 特征 | 量化描述 |
|---|---|---|
| **1** | 浅回调 | 左侧高点到底部的回撤幅度 8%~20%（太深=杯柄，太浅=扁平基部） |
| **2** | 碟底平坦 | 最低点周围 ±5 天（或 ±1 周）振幅 < 5%，碟底不是"一个点"而是"一个区间" |
| **3** | 成交量极度萎缩 | 碟底区间日均量 < 50 日均量 × 0.50（比任何其他形态都严格） |
| **4** | 弧线圆润 | 下行段日收益率标准差 ≤ 0.008，三段斜率逐段递减（越近底部越平缓） |
| **5** | 足够时长 | 左侧高点至今 ≥ 7 周（约 35 个交易日），碟形需要时间"磨"出来 |
| **6** | 突破放量 | 突破碟口上沿当日成交量 ≥ 50 日均量 × 1.3 |

## 4. 量化计算规则

### 4.1 分析周期策略

**核心策略：周线主分析 + 日线精细化。**

碟形基部时间跨度长（通常 2-6 个月），日线噪声过多，用周线识别形态结构可有效过滤假信号。突破点识别则下钻到日线以获取精确的买入时机。

```
阶段一（周线扫描）：宽松参数快速筛选候选标的
  - 分析对象：周前复权收盘价
  - 波峰波谷检测：局部窗口 13 周
  - 目的：找出"可能形成碟形"的股票池

阶段二（日线确认）：对候选标的精细化验证
  - 分析对象：日前复权收盘价 + 成交量
  - 弧线圆润性、碟底平坦性、成交量萎缩等精确检验
  - 目的：排除假阳性，确认突破信号
```

### 4.2 前置条件

**关键设计决策：碟底驱动算法。** 先找最近的低点（碟底），再从碟底向前回溯找对应的局部高点作为碟口上沿。这样做避免将遥远的历史最高点误判为前高。

> **错误示例**：股票在 150 天前冲高到 50 元，回落到 46 元盘整 3 个月，最近 30 天缓慢滑到 42 元形成碟形。如果取 200 天内绝对最高点（50 元），真正的碟形起点（46 元）会被忽略——回调深度被错误地算成 16% 而非实际的 8.7%。

```
步骤 0.1: 在考察日之前，找到周线级别的低点候选 = 碟底
          使用周线局部极值法：price[week] 是前后各 13 周内的最低收盘价

步骤 0.2: 从碟底向前回溯，找到最近的一个"局部高点" = 左侧高点（碟口上沿）
          "局部高点"定义：该周收盘价是前后各 13 周内的最高收盘价
          约束:
            - 左侧高点距碟底 ≥ 3 周（有足够的下行弧线空间）
            - 左侧高点距碟底 ≤ 26 周（约 6 个月，不能太遥远）
            - (左侧高点 - 碟底) / 左侧高点 ∈ [8%, 20%]
          如果找不到满足条件的左侧高点，取下一个次低点作为碟底重复 b)

步骤 0.3: 验证前置上涨
          在左侧高点之前 120 个交易日内找到最低点作为"前低"
          要求 (左侧高点 - 前低) / 前低 ≥ 30%

步骤 0.4: 当前价格处于上升趋势
          收盘价 > SMA50，或 SMA50 斜率 ≥ 0

步骤 0.5: （可选）大市值偏好
          流通市值 ≥ 200 亿优先；< 100 亿排除（小盘"碟形"往往是庄家圆弧顶出货）
```

### 4.3 碟形基部识别（核心算法）

#### 步骤 1：周线波峰波谷检测

在周线级别上识别波峰和波谷序列，作为碟形基部的候选节点：

```
波谷（低点）条件：
  price[week] ≤ price[week-1] AND price[week] ≤ price[week+1]
  且 price[week] 是过去 13 周内的最低收盘价

波峰（高点）条件：
  price[week] ≥ price[week-1] AND price[week] ≥ price[week+1]
  且 price[week] 是过去 13 周内的最高收盘价
```

从检测到的波谷-波峰序列中，按以下规则配对：

```
候选左侧高点（波峰索引 i）
候选碟底（波谷索引 j，j > i）

需同时满足:
  1. 时间窗口: j - i ≥ 3 周（足够的下行弧线空间）
  2. 深度条件: (price[i] - price[j]) / price[i] ∈ [8%, 20%]
  3. 无中间突破: 从 i 到 j 之间，没有周的收盘价 > price[i] + 3%
     （确保这是一段完整的下行弧线，而非已经突破后跌回来的旧基部）
```

#### 步骤 2：碟底平坦性验证（日线）

碟形基部与杯柄最关键的区别在**碟底不是尖的**——它是一段平坦的区间：

```
步骤 2.1: 以碟底日为中心，取前后各 5 个交易日（共 11 天）作为"碟底区间"
步骤 2.2: 碟底区间振幅 = (区间最高收盘价 - 区间最低收盘价) / 区间最高收盘价
          要求 ≤ 0.05（即 5%）
步骤 2.3: 碟底区间的线性回归斜率 |slope| ≤ 0.0005/天
          → 底部是水平的，不是还在下跌
```

**设计理由**：碟子是"平底"的（不像杯子是尖底）。如果最低点是一个孤立的 V 形尖底，那是杯柄形态或普通回调，不是碟形。

#### 步骤 3：弧线圆润性验证（日线）

碟形基部的价格路径应该是**圆润的弧线**，而非锯齿状的下跌-反弹。这是区分碟形与普通回调的第二个关键量化特征：

```
步骤 3.1: 将 [左侧高点 → 碟底] 的下行段等分为 3 段（各占 1/3 时间）
步骤 3.2: 计算每段的日均跌幅
          要求每段日均跌幅 ≤ 0.004/天（即日均跌 < 0.4%）
步骤 3.3: 三段斜率应呈现"放缓"特征：
          第1段斜率（最陡）≥ 第2段斜率 ≥ 第3段斜率（接近水平）
          → 越接近底部跌得越慢，形成圆润的弧线
步骤 3.4: 整个下行段日收益率绝对值序列的标准差 ≤ 0.008
          → 每日涨跌幅变化不大，没有大起大落
```

| 检测项 | 阈值 | 含义 |
|---|---|---|
| 日均跌幅上限 | ≤ 0.004/天 | 不是急跌 |
| 三段斜率趋势 | 逐段递减 | 弧线在"变平" |
| 日收益率标准差 | ≤ 0.008 | 波动均匀、无锯齿 |

#### 步骤 4：成交量验证（日线）

碟形基部的成交量特征是"底部极度萎缩"——比任何其他形态都要更安静：

```
条件 4.1: 碟底区间的日均成交量 < 50 日均量 × 0.50
          → 底部的量不到均值的一半（杯柄通常要求 < 0.80）

条件 4.2: 下行段后半段的日均成交量 < 下行段前半段的日均成交量 × 0.60
          → 越跌量越小（说明抛压在衰竭，而非加速抛售）

条件 4.3: 碟底区间内，单日成交量超过 50 日均量 1.2 倍的天数 = 0
          → 底部没有任何异常放量日（排除有人在底部砸盘）
```

#### 步骤 5：回升确认（日线）

碟形基部必须已经完成"碟底"并开始回升，而非仍在下跌中：

```
条件 5.1: 碟底距今 ≥ 10 个交易日（回升已确认一段时间）
条件 5.2: 碟底之后，存在 ≥ 3 个交易日的收盘价 > 碟底 × 1.05
          → 股价已从底部反弹至少 5%，且持续了多日
条件 5.3: 回升段（碟底 → 考察日）的线性回归斜率 > 0
          → 确认上升趋势
条件 5.4: 当前收盘价 ≥ 左侧高点 × 0.85
          → 已回升至接近碟口上沿
```

### 4.4 柄部（Handle）检测（可选）

碟形基部可以带柄（Saucer With Handle）也可以不带柄。带柄的形态成功率更高——柄部是突破前最后的清洗动作。

| 检测条件 | 量化规则 |
|---|---|
| **柄部时间** | 1~6 周（5~30 个交易日） |
| **柄部回撤** | 从柄部高点到柄部低点的回调 ≤ 15% |
| **柄部位置** | 柄部最低价 ≥ 碟底 + 0.5 × (左侧高点 - 碟底)，即柄在整个基部的上半部 |
| **成交量萎缩** | 柄部期间日均成交量 < 基部整体日均量 × 0.70 |
| **柄部形态** | 柄部应呈轻微向下倾斜或横向整理，不应向上倾斜（向上倾斜=已经在突破） |
| **10 周均线支撑** | 柄部最低点应在 10 周移动平均线（≈50 日均线）之上 |

### 4.5 突破信号检测

#### 突破判定条件

| 条件 | 量化规则 |
|---|---|
| **价格突破** | 当日收盘价 ≥ 买点价格 |
| **成交量放大** | 当日成交量 ≥ 50 日均量 × 1.3 |
| **收盘强势** | 当日收盘价位于当日价格区间的上 50%（收盘在高位，不是冲高回落） |
| **均线多头** | 股价位于 50 日均线和 150 日均线上方（中期与长期上升趋势确认） |
| **突破日阳线** | 收盘 > 开盘 |

#### 买点计算公式

```
有柄形态: 买点 = 柄部最高价 + 0.01（A 股精度）
无柄形态: 买点 = 左侧高点 + 0.01
```

> **注意**：不使用 +0.3% 的百分比缓冲。IBD 的 "+0.1 美元"是针对美股 $50-$200 价位股票的绝对值，A 股直接用 1 分钱精度即可。

#### 虚假突破排除

以下情况不视为有效突破：
- 突破日或次日盘中跌破买点（以最低价计）
- 突破日成交量低于 50 日均量
- 柄部形成期间成交量放大而非萎缩（说明筹码松动）
- 突破前 5 个交易日内存在个股抛盘日特征（放量下跌）

### 4.6 碟形质量评分

为每个检测到的碟形基部计算**综合强度分数**（0~10 分），辅助仓位决策：

| 加分项 | 分值 | 条件 |
|---|---|---|
| 有时间柄 | +2 | 存在符合规则的柄部 |
| 柄部缩量 | +1 | 柄部期间日均量 < 基部均量 × 0.7 |
| 回撤适度 | +2 | 回撤幅度在 8%~15% 之间（不深不浅） |
| 时间充分 | +1 | 基部时间 ≥ 20 周 |
| 右侧放量 | +2 | 回升段日均量 > 下行段后半日均量 × 1.3 |
| 均线支撑 | +1 | 股价位于 50 周均线（≈250 日均线）之上 |
| 碟底极平 | +1 | 碟底区间振幅 < 3%（特别平坦的加分） |

**信号等级**：
- 8~10 分：🔴 强信号，可正常仓位买入
- 5~7 分：🟡 中等信号，轻仓试探
- 0~4 分：⚪ 弱信号，观望或放弃

## 6. 与 CAN SLIM 其他模块的联动

| 联动模块 | 验证规则 |
|---|---|
| **RS 强度（L 维度）** | 碟形形成期间 RS 可能从 85+ 回落到 65-75——这并非走弱，而是"休息"。突破时 RS_250 ≥ 70 即可接受，若 ≥ 90 则确认已进入主升浪 |
| **机构 A/D 评级（I 维度）** | 基部形成期间若 A/D 评级持续为 A/B，确认机构在吸筹 |
| **大盘环境（M 维度）** | 仅在指数出现追盘日或处于 50 日线上方时提高信号权重；熊市中碟形基部突破可靠性大幅降低 |
| **成交量确认** | 左侧缩量 + 右侧放量 + 突破日放量，三者缺一不可 |

### 与口袋支点的配合

碟形基部形成时间长、底部平坦，内部可能出现**多次口袋支点信号**：

```
抢先信号: 碟底区间末期出现口袋支点 → 抢先入场（比突破碟口早 10-30 天）
确认信号: 突破碟口上沿 → 标准买入点

在回测看板中标注:
  🔵 蓝色三角：碟底附近口袋支点（抢先信号）
  🟡 黄色三角：回升途中口袋支点（跟踪信号）
  🔴 红色三角：突破碟口上沿（确认信号）
```

## 8. 回测与评估

### 8.1 评估指标

| 指标 | 目标值 | 说明 |
|---|---|---|
| 信号准确率 | ≥ 55% | 突破后 20 日内涨幅 ≥ 5% 视为正确 |
| 平均收益 | ≥ 12% | 突破后 60 日平均最大收益 |
| 盈亏比 | ≥ 2.5:1 | 平均盈利 / 平均亏损 |
| 误报率 | ≤ 25% | 突破后 5 日内跌回碟口以下的占比 |
| 大盘股覆盖 | ≥ 80% | 沪深 300 成分股中出现碟形基部的覆盖率 |

### 8.2 参数敏感性分析

上线前应至少回测 5 年历史数据，验证：
- 不同回调深度区间（8%~15% vs 15%~20%）下的信号区分度
- 最小周数参数（7 周 vs 5 周）对检出率的影响
- 带柄碟形 vs 无柄碟形的表现差异
- 突破量比阈值（1.1~1.5）的最优值

### 8.3 与现有形态的交叉验证

```
如果同一只股票在同一天被多个形态引擎识别：
  - 碟形 + 杯柄同时识别 → 优先采用杯柄的突破价格（更标准）
  - 碟形 + 扁平基部同时识别 → 形态可能介于两者之间，标注为"混合基部"
  - 碟形 + 口袋支点同时识别 → 高价值信号（见 §6）
```

## 10. FAQ 与 A 股适配

**Q1：碟形基部在 A 股出现频率如何？**

较杯柄形态更少见。若用欧奈尔原版标准（≥ 7 周），A 股检出量较少。建议两阶段扫描——周线用宽松参数（min_descent_weeks=3）快速筛选候选，日线用严格参数确认。配合 RS 强度、成交量等维度进行交叉验证。

**Q2：碟形基部 VS 平底形态（Flat Base）有什么区别？**

平底形态调整幅度 ≤ 15%，时间 5~7 周，是横盘整理（箱体状）。碟形基部回撤 8%~20%，形态为圆弧而非横盘，时间更长（7 周以上）。识别时需注意区分——碟形是"弧线"而非"横线"。

**Q3：碟形基部 VS 杯柄形态如何区分？**

核心区分点在碟底平坦性：碟形底是平的（碟底区间振幅 < 5%），杯柄底是尖的（V 形低点）。如果最低点周围 ±5 天的振幅 > 5%，且回调深度 > 20%，应归为杯柄形态。

**Q4：如何优化识别的准确性？**

两阶段扫描策略（周线宽松筛选 → 日线严格确认）是关键。结合 RS 强度、成交量、机构 A/D 评级等多维度交叉验证，可有效降低假阳性率。

**Q5：A 股小盘股出现"碟形"怎么处理？**

小盘题材股（< 100 亿）的"碟形"往往是庄家对倒出货形成的圆弧顶，而非机构吸筹。建议严格过滤小市值标的，优先在沪深 300、中证 500 成分股中寻找。

**Q6：碟形基部的突破往往不够猛烈怎么办？**

这是正常现象——碟形通常是大盘股，单日涨幅一般不超过 5%。评估突破质量时更应关注成交量的持续性（突破后 3-5 天能否维持放量），而非单日涨幅。止损应设在左侧高点下方 8%。
