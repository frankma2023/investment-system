# 指数强度计算产品需求文档

## 1. 产品概述

本产品旨在对中证指数体系（全市场指数、一级行业、二级行业、主题指数、策略指数）进行多周期相对强度（RS）计算，并通过分层筛选逻辑，识别不同时间维度下的强势指数池，同时提供板块内部个股共振验证功能，为投资决策提供自上而下的量化依据。

## 2. 数据输入

### 2.1 指数日线数据
- 来源：lixinger.db
- 字段：指数代码、交易日期、收盘价、开盘价、最高价、最低价、成交量

### 2.2 指数分类清单
- 已提供文件 `config/index_rs.yaml`，内含5个类别的指数代码及元信息：
  - `market`：29个全市场指数
  - `sector_L1`：11个一级行业指数
  - `sector_L2`：45个二级行业指数
  - `thematic`：229个行业主题指数
  - `strategy`：93个策略指数

### 2.3 个股数据（用于内部共振验证）
- 字段：个股代码、所属行业（申万或中证行业分类）、收盘价、成交量
- 时间范围：与指数同步
- 暂时还没有计算个股的RS值，该指标可暂时预留字段，待未来完善个股数据后补上

## 3. 核心计算模块

### 3.1 多周期收益率计算
对每个指数，分别计算以下回看周期的区间收益率（使用前复权收盘价）：

| 周期标识 | 回看天数 | 用途 |
|---------|---------|------|
| `RET_20` | 20个交易日 | 短期动量 |
| `RET_60` | 60个交易日 | 中期趋势 |
| `RET_120` | 120个交易日 | 长期趋势 |
| `RET_250` | 250个交易日 | 年度趋势 |

计算公式（避免未来函数）：
```
RET_N = (P_today / P_{t-N}) - 1
```
其中 P_today 为最新收盘价，P_{t-N} 为N个交易日前的收盘价。若数据不足N天，则该周期RS不计算或标记为NaN。

### 3.2 相对强度（RS）计算
**重要：RS是在同一指数分类池内部独立计算的百分比排名，不可跨池比较。**

对每个池子（market, sector_L1, sector_L2, thematic, strategy），分别执行以下步骤：

1. 收集池内所有指数在某一周期 `RET_x` 的值。
2. 剔除RET_x为NaN的指数。
3. 对剩余指数按RET_x从大到小排序，计算百分位排名：
   ```
   百分位 = (该指数收益率高于其它指数的个数) / (总指数个数-1) * 100
   ```
4. RS评级 = 四舍五入取整(百分位)，取值范围 0-99。
   - 例如：在29个全市场指数中，若某指数收益率排第2，则高于27个指数，百分位=27/28*100≈96.4，RS=96。

输出字段：
- `RS_20`：基于RET_20的RS评级
- `RS_60`：基于RET_60的RS评级
- `RS_120`：基于RET_120的RS评级
- `RS_250`：基于RET_250的RS评级

### 3.3 池内TOP10指数
对每个池子，按 `RS_120`（默认长期）降序排列，输出前10名指数及其完整指标（代码、名称、各周期RS值、最新价、涨跌幅）。也可支持按 `RS_60` 或 `RS_20` 排序，作为可选项。

## 4. 交叉筛选逻辑：三层强势池

基于各池子的RS评级，定义三个层级的强势池，用于快速定位市场领导力量。

### Level 1：绝对强势池 (The Leading Group)
**定义**：在多个时间维度上均表现最强的指数，反映长期领导板块。

**筛选条件**（需同时满足）：
- `RS_120 >= 90`
- `RS_250 >= 85`
- `RS_60 >= 80`

### Level 2：短期爆发池 (Momentum Breakout)
**定义**：短期动能极强，但长期趋势尚未完全主升，可能处于加速阶段。

**筛选条件**（需同时满足）：
- `RS_20 >= 95`（短期极度强势）
- `RS_60 >= 85`
- `RS_120 >= 70`（长期强度尚可，不必最高）

**输出字段**：额外计算 `RS_20 - RS_60` 差值（显示加速力度）。

### Level 3：潜在共振池 (Pattern Setup)
**定义**：中期和长期强度已经转强，但短期尚未过热，可能出现趋势确认点。

**筛选条件**（需同时满足）：
- `RS_60 >= 85`
- `RS_120 >= 80`
- `RS_20 < 90`（短期尚未拥挤）
- `当前收盘价` 站上20日均线（可选验证）

**注意**：上述阈值均为默认值，可通过配置文件允许用户调整。

## 5. 内部共振验证（个股与板块一致性）

对于 Level 1 和 Level 2 中筛选出的**一级行业指数（L1）或二级行业指数（L2）**，需要验证其板块内部个股的强势股票数量是否与板块强度共振。

### 5.1 所需数据
- 指数成分股列表（需要指数编制方案或从数据源获取）
- 个股的RS评级（可复用相同算法，在全部A股范围内计算个股RS_120）

### 5.2 共振指标计算
对目标行业指数（例如“全指消费”），执行：

1. 获取该指数全部成分股，计算每只个股的 `RS_120`。
2. 统计成分股中 `RS_120 >= 80` 的个股数量，记作 `strong_stock_cnt`。
3. 计算 **强势股占比** = `strong_stock_cnt / 总成分股数 * 100%`。
4. 计算 **成分股RS中位数** 和 **加权平均RS**（按权重，若无权重则等权）。

### 5.3 共振判断标准

| 共振等级 | 条件 | 含义 |
|---------|------|------|
| 强共振 | 强势股占比 ≥ 40% 且 成分股RS中位数 ≥ 75 | 板块上涨由大量强势股驱动，可信度高 |
| 一般共振 | 20% ≤ 强势股占比 < 40% 且 中位数 ≥ 65 | 部分强势股带动，需进一步甄别 |
| 弱共振（警告） | 强势股占比 < 20% 或 中位数 < 60 | 板块可能由少数权重股虚拉，谨慎参与 |
