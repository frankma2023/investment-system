# 抛盘日

## 1. 抛盘日概述

**抛盘日**是指机构投资者系统性卖出导致的市场弱势交易日，表现为：**成交量放大 + 价格无力上涨（或下跌）**。它反映了卖方力量主导市场，买盘无法推动价格上涨。

核心是判断市场顶部和趋势反转，识别**机构抛售行为**而非散户恐慌。

- 统计基准指数采用中证全指（000985），健康度评分也是以中证全指为准。
- 辅助指数为沪深300（000300）和中证500（000905），计算抛盘日数量，但不参与健康度评分。
- 如果3个指数都计算出同一个抛盘日，特别标注出来。


## 2. 抛盘日类型

### 2.1 标准抛盘日

**定义：**
满足以下条件计为一个标准抛盘日：

| 条件 | 量化规则 |
|------|--------|----------------|
| 价格条件 | 收盘价 < 前一日收盘价 | 
| 成交量条件 | 当日成交量 ≥ 前一日 x 1 | 
| 幅度条件 | 当日涨跌幅 ≤ -0.1%（避免平盘或微跌0.00%被误计） |

> 欧原意为当日成交量 > 前一日，我修改为 ≥ 前一日  ，我希望这个计算更敏感些，个人修改了条件。
> 成交量是指成交笔数，而非成交额

### 2.2 特殊抛盘日（假阳线/滞涨）

同时满足以下4个条件则计为1个特殊抛盘日（或叫假阳线抛盘日）。

| 条件 | 量化规则（AND） |
|------|----------------|
| 收盘涨幅 | < 0.2% 且 > −0.3% |
| 盘中最高涨幅 | ≥ 0.5%（确保有冲高动作） |
| 成交量 | > 前一日 × 1.1 |
| 上影线 | ≥ 实体 × 1.5 或 ≥ 振幅 × 0.5 |

> “当日最高价涨幅 ≥ 0.5%”是为了：**排除开盘即横盘、全天无冲高动作的“假信号”**。如果没有这个条件，一个全天在±0.1%内窄幅波动、收盘微涨0.15%、成交量略放大1.3倍、但上影线并不长的交易日，也可能被误判为“特殊抛盘日”。而欧奈尔定义这类抛盘日的核心是：**早盘或盘中曾有冲高（诱多），随后放量回落**，说明机构在上涨中悄悄派发。如果全天根本没有冲高，那就只是普通滞涨，不应计入抛盘日。
> 欧原意成交量> 前一日 × 1.3，但在A股11年内统计1个也没有，故放松条件。
> 欧原意收盘涨幅 < 0.2%，这次也放宽到< 0.2% 且 > −0.3% 

### 2.3 盘中反转抛盘日

同时满足以下4个条件计为1个盘中反转抛盘日。

| 条件 | 量化规则 |
|------|--------|
| 价格条件1 | 收盘价 < 开盘价 （不限制跌幅大小） |
| 价格条件2 | 盘中最高价 ≥ 前一日收盘价 × 1.005（出现过冲高） |
| 成交量 | > 前一日 × 1.1 | 
| K线形态 | 上影线 ≥ 实体 × 1.5 且 收盘价 ≤ (当日最高+最低)/2 |

> 欧原意是成交量 > 前一日 × 1.2，我调成> 前一日 × 1.1，我希望这个计算更敏感些，个人修改了条件。
> 盘中反转抛盘日可与标准抛盘日叠加吗？——不能，同一天只计1个抛盘日，按优先级：反转日 > 特殊抛盘日 > 标准抛盘日。

## 3. 抛盘日的计数规则

| 项目 | 规则 |
|--------|----------------|
| 幅度过滤 | 涨跌幅绝对值 < 0.05% 的“平盘日”不计为任何抛盘日 |
| 重抛盘日多重计数规则 | 若指数单日跌幅 ≥ **-1.5%** 且放量，识别为重抛盘日，并计为 **2个抛盘日**（即权重为2） |
| 确认日抵消机制 | 升势确认日：涨幅 ≥ 1.5% 且成交量 ≥ 前一日 → 可抵消 **1个** 抛盘日（按时间顺序，抵消时间最早的那个），确认日当天不计抛盘日 |
| 窗口滚动 | 严格滚动 25 个交易日，超过25天的抛盘日自动移出统计 |
