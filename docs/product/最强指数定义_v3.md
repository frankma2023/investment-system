# 最强指数

## 一、核心概念

**最强指数**对应的是L（领军股）维度——不仅要选最强的个股，也要选最强的板块/指数。板块效应是筛选牛股最核心的过滤器之一。历史数据显示，股价上涨动力中，50%来自行业趋势，37%来自大盘趋势，只有12%来自个股本身。


## 二、指数RS强度

### 2.1 多周期RET收益率计算

对每个指数，分别计算以下回看周期的区间收益率（使用前复权收盘价）：

| 周期标识 | 回看天数 | 用途 |
|---------|---------|------|
| `RET_20` | 20个交易日 | 短期动量 |
| `RET_60` | 60个交易日 | 中期趋势 |
| `RET_120` | 120个交易日 | 长期趋势 |
| `RET_250` | 250个交易日 | 年度趋势 |

计算公式（避免未来函数）：
```
RET_N = (P_today / P_{t-N}) - 1
```
其中 P_today 为最新收盘价，P_{t-N} 为N个交易日前的收盘价。若数据不足N天，则该周期RS不计算或标记为NaN。

### 2.2 相对强度（RS）计算
**重要：RS是在同一指数分类池内部独立计算的百分比排名，不可跨池比较。**

对每个池子（market, sector_L1, sector_L2, thematic, strategy），分别执行以下步骤：

1. 收集池内所有指数在某一周期 `RET_x` 的值。
2. 剔除RET_x为NaN的指数。
3. 对剩余指数按RET_x从大到小排序，计算百分位排名：
   ```
   百分位 = (该指数收益率高于其它指数的个数) / (总指数个数-1) * 100
   ```
4. RS评级 = 四舍五入取整(百分位)，取值范围 0-99。
   - 例如：在29个全市场指数中，若某指数收益率排第2，则高于27个指数，百分位=27/28*100≈96.4，RS=96。

输出字段：
- `RS_20`：基于RET_20的RS评级
- `RS_60`：基于RET_60的RS评级
- `RS_120`：基于RET_120的RS评级
- `RS_250`：基于RET_250的RS评级


## 三、A/D线

### 3.1 什么是A/D线（累积/派发线）？

A/D线（Accumulation/Distribution Line）是由Marc Chaikin提出的技术指标，用于衡量一段时间内资金的净流入/流出。在欧奈尔体系中，它常被用来辅助验证指数或个股的机构吸筹/出货行为。

**核心思想**：将成交量和收盘价在当日价格区间的位置结合起来，判断当日是买方主导还是卖方主导，然后累加形成一条趋势线。A/D线斜率 > 0 表示资金在该指数上持续净流入。斜率越大，资金流入越强劲。在指数筛选中，优先选择RPS高且A/D线斜率为正的指数作为配置方向。

## 3.2 A/D线计算公式

### 3.2.1 日资金流向乘数（Money Flow Multiplier, MFM）

首先计算当日资金流向乘数，取值范围在 -1 到 +1 之间：

```
MFM = ((收盘价 - 最低价) - (最高价 - 收盘价)) / (最高价 - 最低价)
```

**化简后**：
```
MFM = (2 × 收盘价 - 最高价 - 最低价) / (最高价 - 最低价)
```

**解读**：
- MFM > 0 → 收盘价位于当日区间上半部，买方占优
- MFM < 0 → 收盘价位于当日区间下半部，卖方占优
- 极端情况：收盘价 = 最高价时，MFM = +1（最强买方）
- 极端情况：收盘价 = 最低价时，MFM = -1（最强卖方）

**特殊处理（一字板）**：
当最高价 = 最低价（涨停/跌停一字板）时，MFM = 0？ 但这样会丢失信息。改进方法：
```
if 最高价 == 最低价:
    MFM = (收盘价 / 昨日收盘价) - 1   # 涨幅正负决定MFM符号
```

---

### 3.2.2 资金流量成交量（Money Flow Volume, MFV）

```
MFV = MFM × 当日成交量
```

### 3.2.3 A/D线（累积/派发线）

```
A/D线_t = A/D线_{t-1} + MFV_t
```

初始值通常设为 0（或第一天的MFV）。

---

## 3.3 AD斜率 > 0 的多种计算方法

“斜率 > 0”的核心含义是：**A/D线在一段时间内呈上升趋势**。以下是四种常用计算方法，由简到繁：

### 计算方法一：首尾对比（最简单）

```
斜率方向 = A/D线_t > A/D线_{t-N}
```

其中 N 通常取 5、10 或 20 日。

```python
def is_slope_positive_by_compare(ad_line, lookback=10):
    """方法一：首尾对比"""
    if len(ad_line) < lookback + 1:
        return False
    current = ad_line.iloc[-1]
    past = ad_line.iloc[-lookback-1]
    return current > past
```

**优点**：简单直观  
**缺点**：对单日异常值敏感，若第N日恰好是异常点，可能误判

---

### 计算方法二：线性回归斜率（最精确）

对A/D线的近期数据点进行线性回归，计算回归系数（斜率）。

```python
import numpy as np
from scipy import stats

def calculate_slope(ad_line, lookback=10):
    """方法二：线性回归斜率"""
    if len(ad_line) < lookback:
        return 0
    
    y = ad_line.iloc[-lookback:].values
    x = np.arange(len(y))
    
    # 线性回归：slope = cov(x,y) / var(x)
    slope, intercept, r_value, p_value, std_err = stats.linregress(x, y)
    return slope

def is_slope_positive_by_regression(ad_line, lookback=10, min_slope_threshold=0):
    """判断斜率 > 阈值"""
    slope = calculate_slope(ad_line, lookback)
    return slope > min_slope_threshold
```

**优点**：平滑稳定，不受单日异常值影响  
**缺点**：计算稍复杂

### 实战推荐：综合判断方法

在实际应用中，建议**组合使用**方法一和和方法二：

```python
def is_ad_line_bullish(ad_line, lookback=10, min_slope=0):
    """
    综合判断A/D线是否向上
    
    满足以下任一条件即视为向上：
    1. 首尾对比：当前值 > N日前值
    2. 回归斜率：线性回归斜率 > min_slope
    """
    if len(ad_line) < lookback + 1:
        return False
    
    # 条件1：首尾对比
    condition1 = ad_line.iloc[-1] > ad_line.iloc[-lookback-1]
    
    # 条件2：斜率
    slope = calculate_slope(ad_line, lookback)
    condition2 = slope > min_slope
    
    return condition1 or condition2
```
## 3.4 指数A/D线计算的特殊考虑

### 指数A/D线的归一化

不同指数的A/D线绝对量级差异很大（沪深300 vs 科创50），建议：
- 横向比较时，使用标准化A/D线：`(当前值 - 过去N日均值) / 过去N日标准差`
- 或者仅用于判断自身趋势方向，不做横向比较

## A/D线斜率的评价

在欧奈尔指数筛选体系中，A/D线斜率通常与以下指标配合使用：

| 判断维度 | 条件 | 信号强度 |
| :--- | :--- | :--- |
| 强势信号 | RPS ≥ 90 且 A/D线斜率 > 0 | 🔴 强烈 |
| 弱势信号 | RPS ≥ 90 但 A/D线斜率 ≤ 0 | 🟡 警告（量价背离） |
| 吸筹信号 | A/D线斜率 > 0 且 价格在50日线上方 | 🔴 买入确认 |
| 出货信号 | A/D线斜率 < 0 且 价格跌破50日线 | 🟢 卖出预警 |
