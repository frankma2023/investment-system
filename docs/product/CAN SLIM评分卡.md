### 三、N – 新事物 / 股价新高（满分14分）

#### 形态突破评分细则

时间衰减系数：

| 信号日期 | 系数 |
|----------|:--:|
| 今天 | ×1.0 |
| 昨天 | ×0.8 |
| 2天前 | ×0.6 |
| 3~5天前 | ×0.4 |

引擎基础分（按信号含金量）：

| 引擎 | 基础分 | 说明 |
|------|:--:|------|
| 标准突破（breakout） | 5 | 峰谷结构突破，O'Neil体系核心信号 |
| 口袋支点（pocket_pivot） | 4 | 欧奈尔经典买点 |
| 双重底（double_bottom） | 3 | 底部反转形态 |
| 扁平基部（flat_base） | 3 | 基部突破形态 |
| K线形态·看涨（cdl bullish） | 2/个 | 吞没/锤子/晨星等，累加上限 4 分 |
| TA-Lib·看涨（talib bullish） | 1/个 | 金叉/放量/RSI超卖等，累加上限 3 分 |

> 看跌信号不扣分，仅在综合评语中提示风险。

示例——某股近5天信号：

| 日期 | 信号 | 基础分 | 系数 | 小计 |
|------|------|:--:|:--:|:--:|
| 今天 | 口袋支点 | 4 | ×1.0 | 4.0 |
| 昨天 | K线·看涨吞没 | 2 | ×0.8 | 1.6 |
| 3天前 | TA-Lib·MACD金叉 | 1 | ×0.4 | 0.4 |
| **合计** | | | | **6.0 分** |
