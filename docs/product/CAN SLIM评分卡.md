### 三、N – 新事物 / 股价新高（满分14分）

| 子项 | 量化规则 | 得分 |
| :--- | :--- | :--- |
| **股价接近52周高点** | (当前收盘 − 52周最高) / 52周最高 × 100% | |
| ‣ > −5%（距高点5%以内） | | 7 |
| ‣ −15% ~ −5% | | 5 |
| ‣ < −15% | | 0 |
| **形态突破**（对接6引擎扫描，回溯5个交易日） | Σ(引擎基础分 × 时间衰减系数)，上限14 | |
| **N 维度总分**（上限14） | **/14** | |

#### 形态突破评分细则

时间衰减系数：

| 信号日期 | 系数 |
|----------|:--:|
| 今天 | ×1.0 |
| 昨天 | ×0.8 |
| 2天前 | ×0.6 |
| 3~5天前 | ×0.4 |

引擎基础分（按信号含金量）：

| 引擎 | 基础分 | 说明 |
|------|:--:|------|
| 标准突破（breakout） | 5 | 峰谷结构突破，O'Neil体系核心信号 |
| 口袋支点（pocket_pivot） | 4 | 欧奈尔经典买点 |
| 双重底（double_bottom） | 3 | 底部反转形态 |
| 扁平基部（flat_base） | 3 | 基部突破形态 |
| K线形态·看涨（cdl bullish） | 2/个 | 吞没/锤子/晨星等，累加上限 4 分 |
| TA-Lib·看涨（talib bullish） | 1/个 | 金叉/放量/RSI超卖等，累加上限 3 分 |

> 看跌信号不扣分，仅在综合评语中提示风险。

示例——某股近5天信号：

| 日期 | 信号 | 基础分 | 系数 | 小计 |
|------|------|:--:|:--:|:--:|
| 今天 | 口袋支点 | 4 | ×1.0 | 4.0 |
| 昨天 | K线·看涨吞没 | 2 | ×0.8 | 1.6 |
| 3天前 | TA-Lib·MACD金叉 | 1 | ×0.4 | 0.4 |
| **合计** | | | | **6.0 分** |

### 七、综合评分与等级

```
综合评分 = C + A + N + S + L + I （六维满分为 23+17+14+9+21+18=102）
归一化: 综合得分 / 102 × 100（四舍五入到整数）
```

| 综合评分 | 等级 | 操作建议 |
| :--- | :--- | :--- |
| ≥ 85 | **S** | 优先买入，最大仓位 |
| 75 ~ 84 | **A** | 可以买入，正常仓位 |
| 65 ~ 74 | **B** | 选择性买入，轻仓 |
| 55 ~ 64 | **C** | 少量尝试，需进一步研究 |
| 45 ~ 54 | **D** | 回避 |
| < 45 | **E** | 坚决回避 |

### 八、M – 市场方向（仓位管理因子，不参与评分）

M 因子参考已开发的「大盘健康度评分」（`market_health_daily`），而非单独计算抛盘日/均线位置。用法：

| 大盘健康度 | 建议仓位上限 |
| :--- | :--- |
| ≥ 70（强势） | 满仓 |
| 50 ~ 69（中性） | 七成仓 |
| 30 ~ 49（偏弱） | 半仓 |
| < 30（弱势） | 三成仓或空仓 |

### 十、回测看板设计

每个大维度对应一个可折叠参数卡片，用户可微调阈值并保存。

#### 页面布局

```
┌─ top-nav ─────────────────────────────────┐
├─ top-bar: [股票代码] [评分日期] [计算] ────┤
├─ chart: 雷达图（C/A/N/S/L/I 六维）────────┤
├─ bottom-layout ───────────────────────────┤
│  ┌─ bottom-left (参数卡片) ─┐  ┌─ right ─┐ │
│  │ 🔴 C-当期盈利    [展开]  │  │ 评分明细 │ │
│  │   EPS增速阈值: [──●──] 25%│  │ C: 19/23 │ │
│  │   营收阈值:    [──●──] 25%│  │ A: 14/17 │ │
│  │   ...                    │  │ N: 11/14 │ │
│  │ 🟠 A-年度盈利    [展开]  │  │ ...      │ │
│  │ 🟢 N-形态突破    [展开]  │  │          │ │
│  │ 🔵 S-供给需求    [展开]  │  │ 综合:77  │ │
│  │ 🟣 L-领军股      [展开]  │  │ 等级:A   │ │
│  │ 🟤 I-机构认同    [展开]  │  │          │ │
│  │ [💾 保存配置]            │  │          │ │
│  └──────────────────────────┘  └──────────┘ │
└────────────────────────────────────────────┘
```

#### 参数卡片内容

| 卡片 | 可调参数 | 默认值 |
|------|------|:--:|
| 🔴 C-当期盈利 | EPS增速阈值 / 营收增速阈值 / 加速度阈值 / 扣非占比阈值 | 25% / 25% / 10% / 90% |
| 🟠 A-年度盈利 | 3年CAGR阈值 / 稳定性阈值 | 25% / 30% |
| 🟢 N-形态突破 | 回溯天数 / 时间衰减系数 / 各引擎基础分 | 5天 / 见上表 |
| 🔵 S-供给需求 | 市值分档阈值 / 量比阈值 / 回购比例 | 50/200/500亿 / 1.5 / 1% |
| 🟣 L-领军股 | RS250阈值 / RS动量阈值 / 行业RS阈值 / 超额收益阈值 | 95 / 见上表 / 80 / 5% |
| 🟤 I-机构认同 | 机构持股阈值 / 机构增数阈值 / 资金流阈值 / 研报数阈值 / 负债率阈值 | 15% / 10家 / 2% / 3家 / 60% |
