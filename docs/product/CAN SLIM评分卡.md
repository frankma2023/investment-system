### 三、N – 新事物 / 股价新高（满分17分）

| 子项 | 量化规则 | 得分 |
| :--- | :--- | :--- |
| **股价接近52周高点** | (当前收盘 − 52周最高) / 52周最高 × 100% | |
| ‣ > −5%（距高点5%以内） | | 6 |
| ‣ −15% ~ −5% | | 4 |
| ‣ < −15% | | 0 |
| **形态突破**（对接6引擎扫描，回溯5个交易日） | Σ(引擎基础分 × 时间衰减系数)，与52周高点分**分离计算** | |
| **N 维度总分**（上限17） | **/17** | |

> **v3.0 变更**：52周高点与形态突破不再共享上限，各自计分后相加，上限 17。

#### 形态突破评分细则

时间衰减系数：

| 信号日期 | 系数 |
|----------|:--:|
| 今天 | ×1.0 |
| 昨天 | ×0.8 |
| 2天前 | ×0.6 |
| 3~5天前 | ×0.4 |

引擎基础分（按信号含金量）：

| 引擎 | 基础分 | 说明 |
|------|:--:|------|
| 基部突破（base_breakout） | 5 | O'Neil 体系核心买入信号，统一替代原标准突破/双重底/扁平基部/碟形基部 |
| 口袋支点（pocket_pivot） | 5 | 欧奈尔经典加仓买点 |
| K线形态·看涨（cdl bullish） | 2/个 | 吞没/锤子/晨星等，累加上限 4 分 |
| TA-Lib·看涨（talib bullish） | 1/个 | 金叉/放量/RSI超卖等，累加上限 3 分 |

> 原标准突破/双重底/扁平基部/碟形基部引擎已废弃——它们的底层结构相同（上涨→前高→回调→谷→回升→突破），区别仅在于回调形状和深度，这些差异不改变"是否突破"的判断，细微形态差异由人工复核。现已统一为 `base_breakout` 引擎。
> 看跌信号不扣分，仅在综合评语中提示风险。
