## 1. 回测页面输出需求

- 使用investment-dashboard-xiaohongshu技能开发看板页面，保持页面样式的统一；
- 期望的页面样式是除顶部的banner，与抛盘日回测页面、追盘日回测页面保持一致，导航栏可以彼此加上各自页面链接；
- 页面左侧是可配置参数卡片，有指数池、时间区间可选，使用日历日期选择器，页面右侧表格，展示计算出的指数池及指数的数据；
- 在左侧面板选择可调整的参数卡片，并点击回测按钮进行计算，
- 右侧表格会显示指数的相关数据，展示计算出的指数池及指数的短中长RS值、类型（即绝对强势、短期爆发、潜在共振或为空）；
- 点击保存配置，本次回测配置组合保存在`config/index_rs.yaml`中；

## 2. 左侧参数可配置卡片

- RS计算周期（默认20/60/120/250）
- 各层级RS筛选阈值共4张卡片，
  - 卡片1：L1绝对强势池可配置参数默认数值：
    - RS_120 >= `90`
    - RS_250 >= `85`
    - RS_60 >= `80`
  - 卡片2：L2短期爆发池可配置参数默认数值：
    - RS_20 >= `95`
    - RS_60 >= `85`
    - RS_120 >= `70`
  - 卡片3：L3潜在共振池可配置参数默认数值：
    - RS_60 >= `85`
    - RS_120 >= `80`
    - RS_20 < `90`
    - 当前收盘价站上`20日`均线（上下限10~60日均线）
  - 卡片4：指数共振判断阈值
    - 强势股定义
      - RS_120 >= `80`
    - 强共振 
      - 强势股占总成分股比 ≥ `40%` 且 
      - 成分股RS中位数 ≥ `75` 
    - 一般共振 
      - 20% ≤ 强势股占总成分股比 < 40% 且 
      - 成分股中位数 ≥ `65`
    - 弱共振 | 
      - 强势股占总成分股比 < `20%` 或 
      - 成分股中位数 < `60 `

### 1.3 共振验证卡片（针对Level 1/Level 2中的行业指数）

**输出格式**：对每个L1/L2强势指数，生成一个“共振验证卡片”，包含：
- 指数名称、RS_120值
- 成分股总数
- 强势股数量及占比
- 成分股RS中位数、平均值
- 共振等级（强/一般/弱）及建议

**示例**：全指能源（000986）
- 指数RS_120 = 94
- 成分股总数：42
- 强势股(RS≥80)数量：19  → 占比 45.2%
- 成分股RS中位数：78
- 共振等级：**强共振** ✅
- 建议：行业内部基础扎实，可优先关注该板块中的龙头个股。

## 2. 回测页面技术实现要求

### 7.1 数据处理流程
1. 计算各指数多周期收益率，存储到宽表（index_daily_ret）。
2. 对每个分类池，计算RS评级，存入 index_rs_rating 表。
3. 执行交叉筛选，生成三层强势池视图。
4. 对于筛选出的行业指数，调用个股RS数据，执行共振验证。
5. 输出结果到前端或生成报表。

### 7.2 数据库表设计（建议）

- `index_daily_ret`：多周期收益率（可物化视图）
- `index_rs_rating`：每日RS评级（代码、日期、池子类型、RS_20/60/120/250）


### 7.3 调度与性能
- 使用Python + Pandas + SQLAlchemy，或直接在数据库中编写存储过程（复杂）。
- 建议每日全量计算RS时采用向量化操作。
- 对229个主题指数和93个策略指数，RS计算仍可实时完成（<1秒）。


### 8. 回测页面卡片分类与展示优化

### 卡片1：全市场指数（29个）
- **特点**：数量适中，用户熟悉度高。
- **推荐排序**：默认按120日RS评级降序，突出当前最强的宽基指数（如中证A500、科创50等）。
- **额外高亮**：可标注“创XX日新高”或“RS>90”标签。

### 卡片2：中证一级行业指数（11个）
- **特点**：数量少，适合**热力图可视化**，也可以表格展示。
- **建议**：使用**色块矩阵**（横向为行业，纵向为不同周期RS评级），一眼看出哪些行业在短中长期都强。
- **示例布局**：
  ```
  行业       20日RS   60日RS   120日RS
  全指能源    95      92       88
  全指材料    78      80       75
  ...
  ```

### 卡片3：中证二级行业指数（45个）
- **特点**：数量稍多，可按**一级行业分组**折叠展示（例如先展开“能源”下的二级，再“材料”下的二级...）。
- **默认排序**：按120日RS从高到低，但允许用户按一级行业筛选。
- **突出功能**：点击二级行业可跳转到对应的主题指数（卡片4）中相关主题。

### 卡片4：行业主题指数（229个）
- **数量庞大**，必须提供**搜索框**和**分类筛选器**（如“新能源”、“半导体”、“消费”、“医药”）。
- **默认展示**：只展示**RS评级≥80**的主题（或排名前30），其余折叠或提供“加载更多”。
- **额外标识**：主题指数常与热门概念相关，可加“热度🔥”标记（基于成交额变化或新闻热度，可选）。

### 卡片5：策略指数（93个）
- **按策略类分组**：红利类、低波类、质量类、成长类、等权类等。使用分组折叠。
- **关注核心**：红利策略指数通常长期走牛，可单独突出“股息率”指标（如果有数据源）。
- **展示方式**：每个策略组内按RS评级排序。
