# 指数机构吸筹/出货评级回测看板产品说明 v1

> 对应页面：`web/index-ad-backtest/`  
> 对应引擎：`src/detectors/index_ad.py`  
> 最后更新：2026-05-06

## 1. 页面总体需求

- 使用**小红书卡片风格**，与抛盘日/追盘日/指数RS强度/拥挤度回测页面一致
- 页面布局：**左侧参数配置面板 + 右侧结果展示区**
- 导航栏与所有回测页面互链，当前页标记 `active`
- 日期选择：日历选择器选单个交易日，默认当天（API自动回退到最近交易日）
- 点击「计算」触发 `/api/index-ad` 请求
- 点击「保存配置」写入 `config/index_ad.yaml`，页面加载时自动恢复上次配置

## 2. 左侧参数配置面板

### 2.0 通用选择器

| 控件 | 类型 | 说明 |
|------|------|------|
| 指数池 | 下拉菜单 | market / sector_l1 / sector_l2 / thematic / strategy |
| 计算日期 | 日历组件 | 默认当天 |

### 2.1 卡片1：计算方法 + 窗口

| 参数 | 控件 | 默认值 | 说明 |
|------|------|--------|------|
| 计算方法 | Radio按钮组 | 原始换手率 | raw（原始换手率） / zscore（Z-score标准化） |
| 窗口天数 | 范围滑块 | 65日 | 20~130日，步长5日 |

Radio按钮选中态：红色边框高亮，与其他卡片风格一致。

### 2.2 卡片2：评级映射参考（只读）

展示 A+ ~ E- 的百分位区间和含义，配有彩色评级徽章。不可手动调整，仅供用户参考。

## 3. 右侧结果展示区

### 3.1 统计卡片

四个卡片横排：

| 卡片 | 说明 |
|------|------|
| 指数总数 | 池内有效指数数量 |
| 净吸筹 | AD_Score ≥ 0 的指数数量（红色字） |
| 净出货 | AD_Score < 0 的指数数量（绿色字） |
| 最强吸筹 | 评级最高的指数名称 |

### 3.2 A/D评级结果表格

按 AD_Line 降序排列，默认显示全部指数。支持点击表头排序。

| 列名 | 数据类型 | 说明 |
|------|---------|------|
| 排名 | Number | 1~N |
| 代码 | String | 可点击弹窗查看成分股 |
| 名称 | String | 指数中文名 |
| AD_Line(换手) | Number | raw方法显示3位小数，zscore显示标准差 |
| 百分位 | Number | 池内百分位排名 |
| 评级 | String | 彩色徽章（A+红 → E-紫） |
| AD_Score | Number | -100~+100连续得分 |
| 含义 | String | 评级对应的中文解释 |

### 3.3 评级颜色体系

| 评级 | 徽章颜色 | 含义 |
|------|---------|------|
| A+ | 深红 #D32F2F | 强烈吸筹 |
| A | 红 #F44336 | 强烈吸筹 |
| B+ | 橙 #FF9800 | 较强吸筹 |
| B | 黄 #FFC107 | 温和吸筹 |
| C+ | 黄绿 #CDDC39 | 买入稍多 |
| C | 绿 #8BC34A | 大致平衡 |
| D+ | 深绿 #4CAF50 | 温和出货 |
| D | 蓝 #2196F3 | 大力出货 |
| E | 靛蓝 #3F51B5 | 强烈出货 |
| E- | 紫 #9C27B0 | 强烈出货（最差） |

### 3.4 成分股弹窗

点击表格中任意指数的代码，弹出模态窗口，显示该指数的全部成分股（代码、名称、权重%）。数据来源 `/api/index-constituents`。与其他回测页面弹窗完全一致。

## 4. 交互流程

```
页面加载 → 从 config/index_ad.yaml 恢复上次的 method 和 window_days
         → 用户选择指数池、日期、调整参数
         → 点击「计算」→ GET /api/index-ad?pool=&date=&window=&method=
         → 渲染统计卡片 + 评级表格
         → 点击指数代码 → 弹出成分股弹窗
         → 点击「保存配置」→ POST /api/config?signal_type=index_ad
```
