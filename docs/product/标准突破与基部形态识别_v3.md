# 标准突破买点（Standard Pivot Point）理论知识总结（完整版）

## 1. 核心定义

标准突破买点是指股价在经过一段时间的基部构建（如杯柄、双底、扁平基部等）后，向上突破关键阻力位（即“枢轴点”）的时刻。这是欧奈尔体系中确定性最高、最适合重仓介入的买入信号。

与“口袋支点”相比，标准突破点更强调形态的完成度和趋势的确认，是主力机构完成吸筹、正式发动行情的发令枪。

> 欧奈尔原话：“正确的买入点，就是股票真正启动的那个点位。你必须学会等待一只股票攀升，在做出初步确认的买入点，开始交易。”

---

## 2. 基部形态分类

在识别标准突破买点前，首先需要确认基部形态有效。欧奈尔体系中的主要基部形态：

### 2.1 杯柄形态（Cup with Handle）

| 特征 | 量化规则 |
|------|----------|
| **杯体深度** | 从杯体左高点到杯底，跌幅 20%-40%（熊市中可达50%） |
| **杯体形状** | U形，底部圆润（避免V形） |
| **柄部位置** | 位于杯体右壁的上1/3区域 |
| **柄部回调** | 幅度 ≤ 杯体深度的 1/3 至 1/2，通常 8%-15% |
| **柄部形态** | 向下倾斜或水平，至少2周 |
| **柄部时长** | 最佳 1-4 周，过短洗盘不充分，过长动能消散 |
| **突破点** | 突破柄部最高点（同时也是杯体最高点） |

**量化补充**：
```
Handle_Depth_Ratio = (Cup_Top - Handle_Low) / (Cup_Top - Cup_Bottom)
若 Ratio > 0.66 → 柄部回撤过深，标记为高风险
若 Ratio < 0.33 → 柄部过浅，洗盘可能不充分
```

### 2.2 双底形态（Double Bottom）

| 特征 | 量化规则 |
|------|----------|
| **两个底部** | 两个低点价格相近，差异 < 5% |
| **右底位置** | 右底最好高于左底（强势特征） |
| **中间高点** | 两底之间的反弹高点，比低点高 > 10% |
| **形态时长** | 通常 6-12 周 |
| **突破点** | 突破中间高点 |

### 2.3 扁平基部（Flat Base）

| 特征 | 量化规则 |
|------|----------|
| **形态时长** | 至少 6-7 周 |
| **回调幅度** | 5%-15%（非常紧凑） |
| **波动特征** | 整体波动幅度 < 10%-15%，布林带带宽极度收缩 |
| **优势** | 清洗浮筹极其彻底，爆发力往往强于杯柄 |
| **突破点** | 突破基部最高点 |

### 2.4 碟形基部（Saucer Base）

| 特征 | 量化规则 |
|------|----------|
| **形态形状** | 缓慢的圆弧底，没有明显的柄部 |
| **形态时长** | 通常 7-8 周或更长 |
| **深度** | 较浅，通常 15%-20% |
| **突破点** | 突破圆弧右侧的高点 |

### 2.5 VCP形态（波动率收缩）

| 特征 | 量化规则 |
|------|----------|
| **收缩阶段** | 2-4 次波动率收缩 |
| **每次收缩** | 波动幅度递减 > 20% |
| **突破点** | 最后一次收缩后的突破 |

## 3. 三大核心要素（The Holy Trinity）

一个有效的标准突破必须同时满足以下三个条件：

### 3.1 形态结构

股价必须处于一个清晰的、经过时间换空间的建设性基部中。

**关键特征**：
- 基部内的波动应逐渐收敛
- 成交量在回调时萎缩，显示抛压减轻
- 基部长度至少 5 周（第一阶段基部至少 6 周）

### 3.2 枢轴点

| 维度 | 规则 |
|------|------|
| **定义** | 基部结构中需要突破的关键阻力价位 |
| **杯柄** | 柄部区域的最高点 |
| **双底** | 两个底部之间反弹的高点 |
| **扁平基部** | 盘整区间的最高价 |
| **操作区间** | 枢轴点 +0% 至 +3%（理想买入区域） |
| **埋伏点位** | 枢轴点下方 1%-2% 处提前埋伏（预防假突破或买不进） |

### 3.3 成交量验证（核心）

**黄金法则**：突破日的成交量必须显著放大，代表机构资金大举入场。

| 场景 | 成交量特征 | 解读与操作 |
|------|------------|------------|
| **理想突破** | 1.5x - 2.5x 50日均量 | ⭐⭐⭐⭐⭐ 强力买入，机构共识度高 |
| **前置吸筹突破** | 1.2x - 1.4x 50日均量 | ⭐⭐⭐ 可接受。若突破前10日内出现过口袋支点，说明筹码已被锁定，抛压轻 |
| **天量滞涨（危险）** | > 3.0x - 4.0x 50日均量 且 涨幅 < 2% | ❌ 警惕陷阱。可能是主力借突破出货，若收长上影线，坚决不买 |
| **无量突破** | < 1.2x 50日均量 | ❌ 无效突破，大概率是假突破，随后会跌回基部 |

**最低底线**：成交量 > 50日均量 × 1.4（140%），否则视为弱势突破。

## 4. 价格突破条件

| 条件 | 量化规则 | 说明 |
|------|----------|------|
| **盘中突破** | 当日最高价 > 枢轴点 | 确认突破动作 |
| **收盘确认** | 当日收盘价 > 枢轴点 | 确认突破有效性 |
| **突破幅度** | (收盘价 - 枢轴点) / 枢轴点 ≤ 5% | 过高则不稳健，可能已错过最佳买点 |
| **K线形态** | 收盘 > 开盘（阳线） | 要求阳线 |
| **收盘位置** | 收盘价位于当日区间上半部（> 60%） | 确保上涨有力，非尾盘偷袭 |

## 5. 柄部质量诊断（针对杯柄形态）

柄部是基部的“最后清洗”，其质量直接决定突破成功率：

| 维度 | 优质特征 | 劣质特征 |
|------|----------|----------|
| **回撤深度** | 杯体深度的 1/3 至 1/2 | 超过杯体深度的 2/3，甚至跌回杯底 |
| **持续时间** | 1-4 周（5-20个交易日） | < 3天（洗盘不充分）或 > 4周（动能消散） |
| **内部量能** | 成交量明显萎缩 | 下跌时放量，抛压沉重 |
| **形态方向** | 向下倾斜或水平 | 向上倾斜（高位整理不充分） |

**量化公式**：

```python
def diagnose_handle(cup_top, cup_bottom, handle_low):
    cup_depth = cup_top - cup_bottom
    handle_drop = cup_top - handle_low
    ratio = handle_drop / cup_depth
    
    if ratio > 0.66:
        return "HIGH_RISK", "柄部回撤过深，失败率高"
    elif ratio < 0.33:
        return "LOW_RISK", "柄部回撤过浅，洗盘可能不充分"
    else:
        return "IDEAL", "柄部回撤适中，质量良好"
```

## 5. 风险预警：识别假突破（Bull Traps）

遇到以下情况，坚决不买或立即止损：

| 风险特征 | 描述 | 应对策略 |
|----------|------|----------|
| **大盘背离** | 个股突破，但大盘指数当天大跌或处于明确下降趋势 | ❌ 放弃操作。覆巢之下无完卵 |
| **尾盘回落** | 盘中突破枢轴点，但收盘价回落至枢轴点下方，留长上影线 | ❌ 视为失败。若已买入，次日开盘无条件止损 |
| **天量滞涨** | 成交量 > 3x 50日均量，但涨幅 < 2% | ❌ 可能是主力出货，坚决不买 |
| **行业弱势** | 个股所属板块指数表现疲软，排名后50% | ⚠️ 谨慎。优先选择行业排名前20%的个股 |
| **基本面瑕疵** | 营收/利润增速放缓，或 RS 评级下降 | ⚠️ 重新评估，突破可能需要基本面支撑 |
| **高位突破** | 已是第4阶段基部（牛市中后期） | ⚠️ 风险较高，严格止损 |
