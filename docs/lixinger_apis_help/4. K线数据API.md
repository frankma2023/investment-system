## 4. K线数据API

### 简要描述
获取K线数据，为历史数据的统计指标，未经过任何处理。

### 请求URL
`https://open.lixinger.com/api/cn/company/candlestick`

### 请求方式
`POST`

### url for vibe coding:
`https://www.lixinger.com/api/open-api/html-doc/cn/company/candlestick`
 
### 请求参数
| 参数名称             | 必选 | 数据类型                     | 说明                                                                 |
|----------------------|------|------------------------------|----------------------------------------------------------------------|
| token                | Yes  | String                       | [我的Token](#)页有用户专属且唯一的Token。                             |
| stockCode            | Yes  | String                       | 请参考 [股票信息API](#) 获取合法的 `stockCode`。<br>⚠️ 仅在请求数据为 date range 的情况下生效。 |
| type                 | No   | String                       | 除复权类型，例如：`"lxr_fc_rights"`。<br>当前支持:<br>- 不复权: `ex_rights`<br>- 理杏仁前复权: `lxr_fc_rights`<br>- 前复权: `fc_rights`<br>- 后复权: `bc_rights` |
| date                 | No   | String: `YYYY-MM-DD` (北京时间) | 信息日期。用于获取指定日期数据。（单日查询）                          |
| startDate            | No   | String: `YYYY-MM-DD` (北京时间) | 信息起始时间。用于获取一定时间范围内的数据。<br>⚠️ 开始和结束的时间间隔不超过10年 |
| endDate              | No   | String: `YYYY-MM-DD` (北京时间) | 信息结束时间。用于获取一定时间范围内的数据。<br>默认值是上周一。       |
| adjustForwardDate    | No   | String: `YYYY-MM-DD` (北京时间) | 前复权指定起始时间点。<br>✅ 需与 `endDate` 一起使用，且 ≥ `endDate`。<br>❗ 获取复权类数据时必须传入；不传时默认值 = `endDate` |
| adjustBackwardDate   | No   | String: `YYYY-MM-DD` (北京时间) | 后复权指定起始时间点。<br>✅ 需与 `startDate` 一起使用，且 ≤ `startDate`。<br>❗ 获取复权类数据时必须传入；不传时默认值 = `startDate` |
| limit                | No   | Number                       | 返回最近数据的数量（当未指定 `date` 或 `startDate/endDate` 范围时生效）。 |

### 返回数据说明
| 参数名称         | 数据类型 | 说明       |
|------------------|----------|------------|
| date             | Date     | 数据时间   |
| stockCode        | String   | 股票代码   |
| open             | Number   | 开盘价     |
| close            | Number   | 收盘价     |
| high             | Number   | 最高价     |
| low              | Number   | 最低价     |
| volume           | Number   | 成交量     |
| amount           | Number   | 金额       |
| change           | Number   | 涨跌幅     |
| to_r             | Number   | 换手率     |
| complexFactor    | Number   | 复权因子   |
