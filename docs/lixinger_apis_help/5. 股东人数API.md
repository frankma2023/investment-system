## 5. 股东人数API

### 简要描述
`获取股东人数数据。`

### 请求URL
`https://open.lixinger.com/api/cn/company/shareholders-num`

### 请求方式
`POST`

### url for vibe coding:
`https://www.lixinger.com/api/open-api/html-doc/cn/company/senior-executive-shares-change`

### 请求参数
| 参数名称   | 必选 | 数据类型             | 说明                                                                 |
|------------|------|----------------------|----------------------------------------------------------------------|
| token      | Yes  | String               | [我的Token](#)页有用户专属且唯一的Token。                             |
| stockCode  | Yes  | String               | 请参考 [股票信息API](#) 获取合法的 `stockCode`。                      |
| startDate  | Yes  | String: `YYYY-MM-DD` (北京时间) | 信息起始时间。用于获取一定时间范围内的数据。<br>⚠️ 开始和结束的时间间隔不超过10年 |
| endDate    | No   | String: `YYYY-MM-DD` (北京时间) | 信息结束时间。用于获取一定时间范围内的数据。<br>默认值是上周一。       |
| limit      | No   | Number               | 返回最近数据的数量（若未指定 endDate，则按此数量从 startDate 向前推）。 |

### 返回数据说明
| 参数名称                     | 数据类型 | 说明           |
|------------------------------|----------|----------------|
| date                         | Date     | 数据时间       |
| total                        | Number   | 股东人数       |
| shareholdersNumberChangeRate | Number   | 股东人数变化比例 |
| spc                          | Number   | 股价涨跌幅     |
