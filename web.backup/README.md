# 欧奈尔投资系统 - 前端Demo

## 概述
基于欧奈尔投资理念的现代化投资分析系统前端界面。本Demo展示了市场扫描模块，包含完整的暗色/亮色主题切换、磨砂玻璃效果、A股配色方案（红涨绿跌）以及交互式ECharts图表。

## 设计特点

### 1. 设计系统
- **科技感设计**: 采用深空黑背景(`#050507`)与翠绿色(`#00d992`)主色调，营造专业、科技的视觉感受。
- **磨砂玻璃效果**: 卡片使用半透明毛玻璃背景(`backdrop-filter: blur(10px)`)与阴影结合，实现层次分明的立体效果。
- **A股配色**: 严格遵循A股市场惯例——红色代表上涨(`#ef4444`)，绿色代表下跌(`#10b981`)。
- **完整主题系统**: 支持暗色/亮色模式一键切换，所有颜色通过CSS变量定义，确保一致性。

### 2. 超越现有示例的改进
- **统一的CSS变量系统**: 相比示例中的硬编码颜色，我们建立了完整的设计token系统，便于维护和扩展。
- **响应式布局**: 移动端优先的响应式设计，支持从手机到4K屏幕的完美显示。
- **组件化设计**: 按钮、卡片、表格、图表容器等都有统一的样式规范。
- **交互细节**: 悬停效果、平滑过渡、加载状态、Toast提示等完整的交互体验。

### 3. 图表功能
- **ECharts K线图**: 完整的蜡烛图显示，包含MA5/MA10/MA20/MA50/MA120/MA250均线。
- **行业热力图**: 可视化展示行业相对强度分布。
- **相对强度曲线**: 展示个股RS20/RS120/RS250相对强度变化。
- **专业特性**: 
  - 十字线贯穿蜡烛和成交量柱状体
  - 显示日期、开盘价、最高价、最低价、收盘价、涨跌额、涨跌幅、成交量、成交额、换手率
  - 坐标轴数字颜色与背景色清晰区隔
  - 红涨绿跌的A股配色方案

## 技术实现

### CSS架构
```css
/* 设计token定义 */
:root {
  --color-bg-primary: #050507;
  --color-text-primary: #f1f5f9;
  --color-bullish: #ef4444;    /* 上涨红色 */
  --color-bearish: #10b981;    /* 下跌绿色 */
  --backdrop-blur: blur(10px);
}

/* 毛玻璃卡片 */
.glass-card {
  background: var(--color-bg-glass);
  backdrop-filter: var(--backdrop-blur);
  border: 1px solid var(--color-border-light);
  border-radius: var(--radius-lg);
}
```

### JavaScript模块化
- `initTheme()`: 主题初始化与切换
- `initKLineChart()`: K线图配置与渲染
- `initIndustryHeatmap()`: 行业热力图
- `initRSChart()`: 相对强度图表
- 模拟数据生成与实时更新

### ECharts配置
专门针对A股市场优化的配置：
- 红涨绿跌的颜色映射
- 均线系统（5日到250日）
- 成交量柱状图同步显示
- 十字线工具提示格式化
